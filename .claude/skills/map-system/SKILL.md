# 地圖系統 - 雙模式探索設計（Roadmap，尚未實作）

> ⚠️ 本 skill 描述 Code Quest RPG 願景的地圖系統，目前 cc-office 未實作。當前專案是 Claude Code web client — 現況見 `project-overview`。

## 設計理念

Code Quest 提供**雙模式探索**，讓玩家根據情境選擇最適合的方式：

1. **2D 俯視地圖模式** - 視覺化探索，適合沉浸式體驗
2. **對話式選單模式** - 快速直達，適合效率導向玩家

## 模式 1：2D 俯視地圖 (Overhead Map)

### 使用時機
- 想要探索整個世界
- 想要查看城鎮/野外的完整佈局
- 想要自由移動到不同場所
- 享受視覺化的 RPG 氛圍

### 視覺設計

城鎮中心示例：
```
┌────────────────────────────────────────┐
│          🏰 城鎮中心 (Town Square)      │
├────────────────────────────────────────┤
│                                         │
│    🏪       🍺       📚                 │
│  商業街    酒館   知識圖書館             │
│                                         │
│    🏛️       🏠       🗡️                │
│  公會大廳  玩家住所  鍛造屋              │
│                                         │
│         [玩家 👤]                       │
│                                         │
│  [方向鍵移動] [Enter 進入] [Q 選單]     │
└────────────────────────────────────────┘
```

### 操作方式

**鍵盤控制**：
- `W/↑`：向上移動
- `A/←`：向左移動
- `S/↓`：向下移動
- `D/→`：向右移動
- `Enter/E`：進入建築
- `Q`：開啟快速選單

**滑鼠控制**：
- 左鍵點擊地面：移動到該位置
- 左鍵點擊建築：進入建築
- 右鍵點擊：顯示資訊
- 滾輪：縮放地圖

### 特點

✅ **直觀的空間感** - 清楚看到建築物的相對位置
✅ **視覺探索樂趣** - Pixel Art 風格的城鎮設計
✅ **自由移動** - 可以隨意走動，發現隱藏細節
✅ **RPG 氛圍** - 傳統 JRPG 的遊戲體驗

## 模式 2：對話式選單 (Dialogue Mode)

### 使用時機
- 明確知道要去哪裡
- 想要快速開始任務
- 不需要視覺探索的情境
- 追求效率的玩家

### 視覺設計

城鎮選單示例：
```
┌────────────────────────────────────────┐
│ 📍 你在城鎮中心。想去哪裡？              │
├────────────────────────────────────────┤
│                                         │
│  1. 🏪 商業街 - 購買道具與裝備          │
│  2. 🍺 酒館 - 接取任務與招募夥伴         │
│  3. 📚 知識圖書館 - 學習新技能           │
│  4. 🏛️ 公會大廳 - 查看成就與排行榜       │
│  5. 🏠 玩家住所 - 休息與管理夥伴         │
│  6. 🗡️ 鍛造屋 - 強化裝備與合成道具       │
│  7. 🌲 前往野外                         │
│                                         │
│  [輸入數字或直接說明你想做什麼]          │
└────────────────────────────────────────┘
```

### 操作方式

**數字選擇**：
- 輸入 `1-7` 選擇對應場所
- 快速直達目的地

**自然語言輸入**：
- 「我想買 HP 藥水」→ 自動前往商業街
- 「我想休息」→ 自動前往玩家住所
- 「開始任務」→ 自動前往酒館
- 「管理 Worktree」→ 自動前往公會大廳

### 特點

✅ **快速直達** - 不需要走路，立即到達
✅ **自然語言支援** - AI 理解你的意圖
✅ **效率優先** - 適合任務導向的玩家
✅ **降低學習曲線** - 新手友好

## 模式切換

### 切換方式

在任何時候按 `M` 鍵即可切換模式：

- **地圖模式** → 按 `M` → **對話模式**
- **對話模式** → 按 `M` → **地圖模式**

系統會記住你的偏好，下次啟動時使用相同模式。

### 設計理由

**為何提供兩種模式？**

1. **彈性**：不同玩家有不同偏好
   - 沉浸型玩家喜歡地圖
   - 效率型玩家喜歡對話

2. **情境適應**：不同情境適合不同模式
   - 探索新區域 → 地圖模式（視覺化）
   - 快速完成已知任務 → 對話模式（效率）

3. **學習曲線**：
   - 新手：對話模式（簡單、有引導）
   - 熟練：地圖模式（沉浸、自由）

4. **自然語言優勢**：
   - 對話模式可以說「我想做什麼」
   - 不需要記住精確的位置
   - 更符合 AI 助手的特性

## CodeLand 世界地圖結構

### 3 大區域

```
┌─────────────────────────────────────────┐
│                                         │
│         🏔️ 副本區域 (Dungeon Zone)      │
│      特殊任務、Boss 戰、完成前無法退出    │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│          🌲 野外區域 (Wilderness)        │
│     危險區域、隨機遭遇戰、執行任務場所    │
│                                         │
├─────────────────────────────────────────┤
│                                         │
│           🏰 城鎮區域 (Town)             │
│      安全區、商店、NPC、休息補給         │
│                                         │
└─────────────────────────────────────────┘
```

### 城鎮區域（安全區）

**5 個主要場所**：

| 場所 | 圖標 | 功能 | 快捷鍵 |
|-----|------|------|--------|
| 商業街 | 🏪 | 7個商店（技能、工匠、MCP、傭兵、寶物、訓練、錢莊） | `1-7` |
| 酒館 | 🍺 | AI對話、接任務（不消耗MP） | `H` |
| 靜止之間 | ⏸️ | Plan Mode、深度思考 | `P` |
| 公會大廳 | 🏛️ | Worktree管理、平行世界 | `G` |
| 玩家住所 | 🏠 | 休息恢復、存檔、設定 | - |

**特性**：
- ✅ 安全區，不會觸發戰鬥
- ✅ HP/MP 快速恢復（2 HP/秒, 1 MP/秒）
- ✅ 可執行 Worktree 操作
- ✅ 所有商店和服務開放

### 野外區域（危險區）

**4 個子區域**：

| 區域 | 等級 | 遭遇率 | 特性 |
|------|------|--------|------|
| 🌳 森林 | Lv.2-6 | 30% | 入門級任務 |
| ⛰️ 山脈 | Lv.7-11 | 50% | 中級任務 |
| 🏜️ 荒野 | Lv.12-15 | 70% | 高級任務 |
| 🌋 火山 | Lv.15+ | 90% | 專家級任務 |

**特性**：
- ⚠️ 危險區，隨機觸發遭遇戰
- ⚠️ HP/MP 不恢復（0.1/秒微量恢復）
- ⚠️ 無法訪問商店
- ⚠️ 無法執行 Worktree 操作

**遭遇戰觸發機制**：
根據 Prompt 複雜度：
- 0-2分：對話型 → 不觸發戰鬥
- 3-7分：簡單任務 → 主目錄同步戰鬥
- 8+分：複雜任務 → Worktree 異步戰鬥

### 副本區域（特殊任務）

**4 個副本**：

| 副本 | 難度 | Boss | 特殊機制 |
|------|------|------|---------|
| 🐛 Bug洞窟 | ⭐⭐⭐ | Bug之王 | 追蹤錯誤日誌 |
| 🏗️ 架構迷宮 | ⭐⭐⭐⭐ | 遺留代碼守護者 | 必須Plan Mode |
| 🧪 測試試煉 | ⭐⭐⭐⭐ | 測試監督者 | 100%覆蓋率 |
| 🔒 安全地牢 | ⭐⭐⭐⭐⭐ | 漏洞注入魔王 | 即死技能 |

**特性**：
- ⚠️⚠️ 進入後無法退出（必須完成或失敗）
- ⚠️⚠️ Boss級戰鬥
- 🎁 豐厚獎勵（EXP/Gold/特殊技能/成就）

## 快捷鍵總覽

### 移動控制
- `W/A/S/D` 或 `↑/←/↓/→`：移動角色
- `E` 或 `Enter`：進入建築
- `M`：切換地圖/對話模式

### 場所快捷鍵
- `1-7`：商店直達（地圖模式）
- `H`：酒館
- `G`：公會大廳
- `P`：靜止之間（Plan Mode）
- `B`：返回城鎮

### 系統快捷鍵
- `Tab`：快速選單（多戰鬥切換）
- `I`：物品欄
- `C`：角色面板
- `Esc`：返回/取消

## 使用建議

### 新手玩家
1. **從對話模式開始** - 有引導、簡單直觀
2. 熟悉各場所功能後，嘗試地圖模式
3. 按 `M` 隨時切換，找到最適合的方式

### 效率玩家
1. **使用對話模式** - 直接說「我想做什麼」
2. 利用自然語言快速達成目標
3. 減少移動時間，專注任務本身

### 沉浸玩家
1. **使用地圖模式** - 享受 Pixel Art 世界
2. 自由探索，發現隱藏細節
3. 使用快捷鍵提高操作效率

### 開發者
- 在城鎮進行 Worktree 操作（公會大廳）
- 野外執行任務會觸發戰鬥
- 複雜任務（8+分）自動創建 Worktree 隔離
