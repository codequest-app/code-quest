## 夥伴技能按鈕

### 可用狀態

```
┌───────────────────────┐
│ 🔍 安全掃描            │
```

**樣式**:
- 背景: 藍色漸層

- 點擊: 縮小 0.95 倍

```
┌───────────────────────┐
│ 🛡️ 守護之盾 (灰色)     │
```

**樣式**:
- 背景: 灰色
- 不可點擊
- 冷卻數字覆蓋層（大字體）

## 夥伴動畫

### 召喚動畫

**時間軸**:
```
0.0s  ├─ 播放召喚音效
0.0s  ├─ 召喚陣出現（從小到大）
0.5s  ├─ 閃光效果
0.7s  ├─ 夥伴從召喚陣中升起
1.0s  ├─ 夥伴落地
1.2s  ├─ 顯示夥伴資訊面板
1.5s  ├─ 播放召喚台詞
2.0s  ├─ 動畫完成
```

**CSS 動畫**:
```css
@keyframes summonAppear {
  0% {
    transform: translateY(100px) scale(0);
    opacity: 0;
  }
  50% {
    transform: translateY(-20px) scale(1.2);
    opacity: 1;
  }
  100% {
    transform: translateY(0) scale(1);
    opacity: 1;
  }
}
```

### 技能施放動畫

**安全掃描動畫**:
```
1. 夥伴向前移動（0.3s）
2. 掃描光波從夥伴發出（0.5s）
3. 光波擊中敵人（0.2s）
4. 敵人閃爍受傷（0.3s）
5. 傷害數字彈出（0.5s）
6. 夥伴返回原位（0.3s）
```

**守護之盾動畫**:
```
1. 夥伴舉起手（0.2s）
2. 護盾能量聚集（0.3s）
3. 護盾展開包圍玩家（0.5s）
4. 護盾閃爍穩定（0.3s）
```

**OWASP打擊動畫**:
```
1. 畫面震動（0.2s）
2. 夥伴蓄力（0.5s）
3. 能量波發射（0.3s）
4. 大範圍爆炸效果（0.7s）
5. 敵人受到重擊（0.5s）
6. 傷害數字特大（1.0s）
```

### 受傷動畫

```css
@keyframes companionHurt {
  0% { transform: translateX(0); }

  50% { transform: translateX(10px); }

}
```

### 倒下動畫

```css

  0% {
    transform: scale(1) rotate(0deg);

  }

    transform: scale(1.1) rotate(-5deg);
```

## 總結

夥伴系統 UI 設計重點：

**✅ 清晰的資訊展示**:
- 夥伴狀態一目了然
- 技能可用性明確
- HP/MP 實時更新

**✅ 流暢的動畫效果**:
- 召喚動畫有儀式感
- 技能動畫有打擊感
- 狀態變化有反饋

**✅ 直觀的操作**:
- 技能按鈕狀態明確
- 懸停提示詳細
- 快捷操作支援

**✅ 響應式適配**:
- 桌面版詳細展示
- 平板版折疊優化
- 手機版精簡呈現
