# 夥伴系統（Companion System）

**依賴**: 戰鬥系統
**被依賴**: 非同步戰鬥系統

---

## 📖 系統概要

夥伴系統將 Subagent 提升為戰鬥夥伴。夥伴擁有獨立的 HP/MP、戰鬥屬性、專屬技能和 AI 行為，可在戰鬥中協同作戰並隨著使用次數成長。不同於召喚獸的短暫爆發，夥伴是長期穩定的戰友，陪伴玩家完成整場戰鬥。

---

## 🎯 核心功能

### 1. Subagent → 夥伴映射

| Subagent 特性 | 戰鬥夥伴呈現 |
|--------------|------------|
| Agent 類型 | 夥伴職業/角色 |
| Agent Memory | 夥伴經驗/成長 |
| Agent Model (sonnet/opus) | 夥伴等級/能力 |
| Agent Tools | 夥伴專屬技能 |
| 持續執行 | 戰場上的持續存在 |
| Agent 完成任務 | 施放技能/攻擊 |

### 2. 戰鬥屬性系統

**基本資訊**：
- 角色名稱和稱號（如 "CodeGuard - 代碼守護者"）
- 頭像 emoji
- 元素屬性

**戰鬥屬性**：
- HP/MP：獨立的生命和魔力值
- 攻擊/防禦/速度/智慧：影響戰鬥效果
- 狀態：standby | active | exhausted | ko

### 3. 夥伴專屬技能

每個夥伴根據其 Agent 特性擁有 3-5 個專屬技能：

**技能類型**：
- **attack**：攻擊技能（造成傷害）
- **support**：支援技能（治療、護盾、Buff）
- **special**：終極技能（強力效果，冷卻長）

**範例 - CodeGuard**：
- 安全掃描（攻擊，對代碼型敵人 +50%）
- 代碼護盾（支援，為玩家提供護盾）
- OWASP 之力（終極，全體敵人掃描並標記弱點）

### 4. 夥伴 AI 系統

夥伴在戰鬥中自動行動，根據戰況選擇最佳技能：

**決策優先級**：
1. 玩家 HP < 30% → 使用治療/支援技能
2. 敵人有弱點 → 使用有效攻擊技能
3. 默認 → 使用基礎攻擊技能

### 5. 成長系統

- **經驗值累積**：戰鬥勝利獲得經驗
- **等級提升**：解鎖新技能、提升屬性
- **使用統計**：記錄出戰次數和勝利次數

## 🔗 與其他系統的關係

### 依賴關係

- **戰鬥系統** → 夥伴參與戰鬥回合
- **商店系統** → 傭兵公會召喚夥伴

### 被依賴關係

- **非同步戰鬥系統** → 後台戰鬥中夥伴自動行動

### 與召喚獸的差異

| 特性 | 夥伴 | 召喚獸 |
|-----|-----|-------|
| 來源 | Subagent | 技能/道具/組合技 |
| 持續時間 | 戰鬥全程 | 1-3 回合 |
| 成長性 | 有經驗值、升級 | 無成長，固定能力 |
| 槽位 | 夥伴槽（最多2個） | 召喚獸槽（最多1個） |
| 用途 | 持續戰鬥支援 | 爆發傷害、緊急支援 |

## 💡 設計要點

### 設計理念

```
夥伴 = 長期戰友
├─ 陪伴整場戰鬥
├─ 穩定輸出/支援
└─ 能力隨時間成長
```

### 關鍵設計決策

1. **Subagent 映射邏輯**
   - Agent Memory → 夥伴經驗
   - Agent Model → 夥伴能力
   - Agent Tools → 專屬技能

2. **夥伴 AI 設計**
   - 自動決策，不增加用戶負擔
   - 根據戰況智能選擇技能
   - 優先保護玩家

3. **成長系統**
   - 使用越多，越強大
   - 鼓勵玩家培養喜歡的夥伴
   - 提供長期目標

## 🚀 快速參考

### 夥伴範例

**CodeGuard (code-guardian)** - 代碼守護者
- 屬性：守護型
- 專長：安全審查、代碼防護
- 技能：
  - 安全掃描（攻擊，對代碼型 +50%）
  - 代碼護盾（支援，護盾 = 防禦 × 2）
  - OWASP 之力（終極，標記所有敵人弱點）

**DevMentor (dev-mentor)** - 開發導師
- 屬性：支援型
- 專長：代碼建議、經驗傳授
- 技能：
  - 知識傳授（支援，提升玩家攻擊 20%）
  - 代碼示範（攻擊，中等傷害）
  - 經驗之光（終極，全隊恢復 MP）

### 召喚流程

```
1. 進入傭兵公會（商業街）
2. 查看可用 Subagents
3. 選擇夥伴召喚（消耗 30-50 MP）
4. 播放召喚動畫
5. 夥伴加入隊伍（最多2個）
6. 戰鬥時自動參與
```

### 夥伴狀態

- **standby**：待命中，未召喚
- **active**：戰鬥中，正常行動
- **exhausted**：疲憊，MP耗盡無法行動
- **ko**：倒下，HP = 0 無法行動
