## 操作方式設計

### 1. 鍵盤操作（PC）

#### 移動控制
```
移動:
W / ↑  - 向上移動
A / ←  - 向左移動
S / ↓  - 向下移動
D / →  - 向右移動

進入建築:
E / Enter  - 進入當前靠近的建築
Esc        - 退出/返回上一級
```

#### 快捷鍵
```
場所快捷鍵:
1  - 技能商店
2  - 工匠鋪
3  - 魔法圖書館
4  - 傭兵公會
5  - 寶物庫
6  - 訓練場
7  - 錢莊

H  - 酒館
G  - 公會大廳
P  - 靜止之間（Plan Mode）
B  - 返回城鎮

系統快捷鍵:
M  - 打開大地圖
Tab  - 快速場所選單
I  - 物品欄
C  - 角色面板
```

---

### 2. 滑鼠操作（PC）

```
點擊:
• 左鍵點擊建築 → 進入建築
• 左鍵點擊地面 → 角色移動到該位置
• 右鍵點擊建築 → 顯示建築資訊

拖曳:
• 拖曳地圖 → 移動視角（僅在大地圖模式）

滾輪:
• 滾輪 → 縮放地圖（僅在大地圖模式）
```

---

### 3. 觸控操作（iPad / iPhone）

```
觸控手勢:
• 點擊建築 → 進入建築
• 點擊地面 → 角色移動到該位置
• 長按建築 → 顯示建築資訊

拖曳角色:
• 拖曳玩家圖標 → 移動角色（實時跟隨手指）

雙指操作:
• 雙指縮放 → 縮放地圖
• 雙指拖曳 → 移動視角

虛擬按鈕:
左下角:
┌─────────┐
│  ↑      │
│ ← ↓ →   │  方向搖桿
└─────────┘

右下角:
┌─────────┐
│   [E]   │  互動按鈕
│ 進入建築 │
└─────────┘

右上角:
┌─────────┐
│ [M] [I] │  地圖/物品
│ [Tab]   │  選單
└─────────┘
```
