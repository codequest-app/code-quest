```

   • 支持文字檔、圖片
   • 自動加入context

按鈕說明:
• [💬自由對話] - 切換為自由對話模式（預設）
• [🔄清除對話] - 清除當前session，重新開始

## UI元素互動細節

### 數值顯示格式規範

**重要**: 為確保 UI 一致性，所有 HP/MP/EXP 顯示必須遵循以下標準格式：

#### 玩家狀態列（頂部狀態欄）

```
┌─────────────────────────────────┐
│ Lv.5 冒險者                      │
│ ❤️ HP: 80/100  ████████░░       │
│ ⚡ MP: 60/100  ██████░░░░       │
│ ⭐ EXP: 300/500 ███░░░░░░░      │
└─────────────────────────────────┘
```

**格式規則**:
- 數值格式: `當前值/最大值`，數字右對齊
- 進度條: 10 格方塊，填充 `█` 表示已滿，`░` 表示空缺
- 圖標在左，數值在中，進度條在右
- 顏色: HP (紅色 #E74C3C), MP (藍色 #3498DB), EXP (金色 #F39C12)

#### 實作範例 (React)

```tsx
// HP/MP 進度條組件
interface ProgressBarProps {
  value: number;
  max: number;
  size?: 'compact' | 'normal';  // 4格或10格
  color: 'hp' | 'mp' | 'exp' | 'enemy';
  showNumbers?: boolean;
}

function ProgressBar({ value, max, size = 'normal', color, showNumbers = true }: ProgressBarProps) {
  const blocks = size === 'compact' ? 4 : 10;
  const filled = Math.floor((value / max) * blocks);
  const empty = blocks - filled;

  const colors = {
    hp: '#E74C3C',
    mp: '#3498DB',
    exp: '#F39C12',
    enemy: '#E67E22'
  };

  return (
    <div className="flex items-center gap-2">
      <span className="text-xs font-pixel">
        {'█'.repeat(filled)}{'░'.repeat(empty)}
      </span>
      {showNumbers && (
        <span className="text-xs font-mono" style={{ color: colors[color] }}>
          {value}/{max}
        </span>
      )}
    </div>
  );
}
```

### 串流渲染優化

```javascript
// 使用requestAnimationFrame優化打字機效果
class TypewriterEffect {
  constructor(element, text, speed = 50) {
    this.element = element;
    this.text = text;
    this.speed = speed;
    this.index = 0;
    this.lastTime = 0;
  }

  animate(timestamp) {
    if (timestamp - this.lastTime >= this.speed) {
      if (this.index < this.text.length) {
        this.element.textContent += this.text[this.index];
        this.index++;
        this.lastTime = timestamp;
      } else {
        return; // 完成
      }
    }
    requestAnimationFrame((t) => this.animate(t));
  }

  start() {
    requestAnimationFrame((t) => this.animate(t));
  }
}
```

## 總結

### 核心設計原則

1. **即時回饋** - 所有操作都有立即的視覺/聽覺回饋
2. **狀態可見** - HP/MP/EXP等狀態隨時可見
3. **減少認知負擔** - 預設技能降低使用門檻
4. **漸進式揭露** - 進階功能隱藏，避免overwhelm
5. **錯誤容忍** - 提供重試和恢復機制
6. **無障礙** - 支援鍵盤、螢幕閱讀器、高對比

### 關鍵互動體驗

- ✨ **施法動畫** - 讓每次技能使用都有儀式感
- 🎵 **音效回饋** - 打字、施法、升級都有音效
- 📊 **數值視覺化** - HP/MP/EXP用進度條顯示
- ⌨️ **快捷操作** - 鍵盤、技能按鈕多種輸入方式
- 🔄 **即時串流** - 打字機效果讓AI回應更生動
- 🎯 **引導式設計** - 新手引導、提示幫助用戶快速上手
