# 商店系統 - UI 設計文檔

**日期**: 2026-02-05
**版本**: v1.0

---

## 設計概覽

### UI 設計原則

1. **一致性**: 所有商店使用統一的設計語言
2. **清晰性**: 資訊層次分明，重點突出
3. **可訪問性**: 支持鍵盤導航和快捷鍵
4. **沉浸感**: RPG 風格的視覺元素
5. **實用性**: 美觀但不犧牲功能

### 視覺風格

**色彩方案**:
- 背景：深色主題（#1a1a1a）
- 主色調：藍紫色（#6366f1）
- 強調色：金色（#fbbf24）
- 成功：綠色（#10b981）
- 警告：橙色（#f59e0b）
- 錯誤：紅色（#ef4444）

**字體**:
- 標題：中文黑體，英文 Inter Bold
- 正文：中文黑體，英文 Inter Regular
- 代碼：JetBrains Mono

**圖標**:
- 使用 Emoji 作為主要圖標（簡潔、跨平台）
- 關鍵操作使用自定義 SVG 圖標

### 交互設計

**點擊商店圖標**:
- 懸停時圖標放大並顯示商店名稱
- 點擊後進入對應商店
- 有新內容時顯示紅點通知

**鍵盤導航**:
- `Tab` 鍵循環選擇商店
- `Enter` 進入選中的商店
- `Esc` 返回探索模式
- `K` 直接打開技能商店
- `G` 直接打開傭兵公會
- `M` 直接打開魔法圖書館

**動畫效果**:
- 進入城鎮時淡入動畫
- 商店圖標懸停時輕微浮動
- 點擊時縮放動畫

### UI 元素說明

**技能卡片狀態**:
- ✅ 已解鎖：顯示綠色勾選，可以使用和升級
- 🔒 未解鎖：顯示鎖圖標，灰色顯示，顯示解鎖條件

**分類標籤**:
- `[官方技能]` - 藍色背景
- `[自定義技能]` - 綠色背景
- `[已解鎖]` - 無特殊標記
- `[未解鎖]` - 半透明顯示

**按鈕樣式**:
- `[使用]` - 主要操作，藍色按鈕
- `[查看詳情]` - 次要操作，灰色按鈕
- `[升級]` - 金色按鈕，顯示升級成本
- `[🔒 解鎖]` - 禁用狀態，顯示解鎖條件

.shop-page {

.shop-page-enter {
  opacity: 0;
  transform: translateY(20px);

  transform: translateY(0);
}

.shop-page-exit {
  opacity: 1;
  transform: translateY(0);
}

.shop-page-exit-active {
  opacity: 0;
  transform: translateY(-20px);
}

}
