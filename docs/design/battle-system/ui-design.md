## UI 組件規格

### 1. 戰鬥主畫面 (BattleScreen)

**功能**:
- 統籌整個戰鬥界面
- 管理子組件
- 處理戰鬥狀態轉換

**布局**:
```
┌─────────────────────────────────────┐
│         背景層 (background)           │
│  ┌─────────────────────────────┐    │
│  │      敵人區域 (enemy-area)    │    │
│  └─────────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐    │
│  │   戰鬥日誌 (battle-log)       │    │
│  └─────────────────────────────┘    │
│                                     │
│  ┌─────────────────────────────┐    │
│  │  行動選單 (action-menu)       │    │
│  └─────────────────────────────┘    │
│                                     │
│  傷害數字層 (damage-numbers)         │
│  勝利畫面層 (victory-screen)         │
└─────────────────────────────────────┘
```

**動畫進入效果**:
```
1. 背景淡入 (fade in, 0.3s)
2. 敵人從上方滑入 (slide down, 0.5s)
3. 戰鬥日誌從下方滑入 (slide up, 0.5s, delay 0.2s)
4. 行動選單從下方滑入 (slide up, 0.5s, delay 0.3s)
```

**Pixel Art 風格**:
- 使用 8-bit 風格字體
- 復古色彩調色板
- 像素化邊框和背景

### 2. 敵人展示 (EnemyDisplay)

**功能**:
- 顯示敵人圖示、名稱、等級
- 顯示 HP 條
- 顯示元素類型
- 提示弱點

**布局**:
```
┌──────────────────────────────┐
│        ┌──────┐              │
│        │ 💻  │ ← 浮動圖示     │
│        │      │              │
│        └──────┘              │
│                              │
│   中等的代碼挑戰   Lv.5       │
│                              │
│   HP: ▓▓▓▓▓▓░░░░  450/600   │
│                              │
│   [Logic] 元素                │
│   弱點: code-reviewer         │
└──────────────────────────────┘
```

**HP 條設計**:
```
正常狀態 (HP > 50%):
▓▓▓▓▓▓▓▓░░░░  (綠色)

警告狀態 (HP 20%-50%):
▓▓▓▓░░░░░░░░  (黃色)

危險狀態 (HP < 20%):
▓▓░░░░░░░░░░  (紅色，閃爍)
```

**浮動動畫**:
```
敵人圖示持續緩慢上下浮動:
- 上升 10px (1s, ease-in-out)
- 下降 10px (1s, ease-in-out)
- 循環播放
```

**受傷動畫**:
```
受到傷害時:
1. 圖示閃爍紅色 (0.1s)
2. 水平震動 (±5px, 0.2s)
3. HP 條平滑減少 (0.5s)
```

### 3. 戰鬥日誌 (BattleLog)

**功能**:
- 顯示戰鬥過程
- 滾動顯示最新消息
- 不同類型消息使用不同顏色

**布局**:
```
┌─────────────────────────────────────┐
│ 戰鬥日誌                             │
├─────────────────────────────────────┤
│ [1] 遭遇敵人: 中等的 Bug 怪物 Lv.5!   │
│ [1] 弱點: debug-helper               │
│ [2] 使用 code-reviewer! 造成 150 傷害!│
│ [2] 有效攻擊!                        │
│ [3] Bug 怪物 反擊! 消耗 25 MP!        │
│ [4] 使用 debug-helper! 造成 270 傷害! │
│ [4] 弱點攻擊! 傷害加成!               │
│ [5] ✨ 勝利!                         │
│ [5] 獲得 EXP +225, Gold +97          │
└─────────────────────────────────────┘
    ▲ 自動滾動到最新
```

**顏色方案**:
```css
.log-encounter    { color: #ffcc00; }  /* 遭遇 - 黃色 */
.log-player       { color: #00ccff; }  /* 玩家攻擊 - 藍色 */
.log-agent        { color: #cc00ff; }  /* Agent 攻擊 - 紫色 */
.log-enemy        { color: #ff6600; }  /* 敵人攻擊 - 橙色 */
.log-critical     { color: #ff0000; font-weight: bold; }  /* 暴擊 - 紅色加粗 */
.log-victory      { color: #00ff00; font-size: 1.2em; }   /* 勝利 - 綠色放大 */
.log-defeat       { color: #999999; }  /* 失敗 - 灰色 */
.log-info         { color: #cccccc; }  /* 提示 - 淺灰 */
.log-reward       { color: #ffd700; }  /* 獎勵 - 金色 */
```

**動畫效果**:
```
新消息出現:
1. 從左側滑入 (translateX: -20px → 0, 0.3s)
2. 透明度淡入 (opacity: 0 → 1, 0.3s)
```

### 4. 傷害數字 (DamageNumber)

**功能**:
- 在敵人位置彈出傷害數字
- 區分普通傷害和弱點傷害

**普通傷害**:
```
       180 ↑
         ↑
         ↑
      (向上飄動，淡出)
```

**弱點傷害**:
```
      270!  ← 更大，帶感嘆號
       ↑
       ↑
       ↑
    (放大，晃動，向上飄動)
```

**動畫參數**:
```javascript
// 普通傷害
{
  initialScale: 1.0,
  finalScale: 1.0,
  duration: 1000ms,
  translateY: -80px,
  color: '#ffffff'
}

// 弱點傷害
{
  initialScale: 1.2,
  finalScale: 1.5,
  duration: 1000ms,
  translateY: -80px,
  color: '#ff0000',
  wobble: true,  // 左右晃動
  bold: true
}
```

**CSS 樣式**:
```css
.damage-number {
  position: absolute;
  font-size: 2em;
  font-weight: bold;
  color: white;
  text-shadow: 2px 2px 4px #000;
  pointer-events: none;
}

.damage-number.weak-hit {
  font-size: 2.5em;
  color: #ff0000;
  animation: wobble 0.3s ease-in-out;
}

@keyframes wobble {
  0%, 100% { transform: rotate(0deg); }
  25% { transform: rotate(-5deg); }
  75% { transform: rotate(5deg); }
}
```

## 音效設計

### 戰鬥音效列表

**UI 音效**:
- `ui_select.wav`: 選擇選項
- `ui_confirm.wav`: 確認操作
- `ui_cancel.wav`: 取消/返回

**戰鬥音效**:
- `attack_normal.wav`: 普通攻擊
- `attack_weak.wav`: 弱點攻擊
- `attack_resist.wav`: 抗性攻擊
- `enemy_hit.wav`: 敵人受傷
- `enemy_counter.wav`: 敵人反擊
- `summon.wav`: 召喚 Agent

**結果音效**:
- `battle_victory.wav`: 戰鬥勝利
- `battle_defeat.wav`: 戰鬥失敗
- `level_up.wav`: 等級提升
- `reward.wav`: 獲得獎勵

**環境音效**:
- `battle_bgm.mp3`: 戰鬥背景音樂 (循環)

### 音效觸發時機

```javascript
// 攻擊音效
onSkillCast() {
  if (isWeakness) {
    playSound('attack_weak');
  } else if (isResistance) {
    playSound('attack_resist');
  } else {
    playSound('attack_normal');
  }
}

// 傷害音效
onDamageDealt() {
  playSound('enemy_hit');
  if (enemy.hp <= 0) {
    setTimeout(() => playSound('battle_victory'), 500);
  }
}

// 反擊音效
onEnemyCounter() {
  playSound('enemy_counter');
}
```
