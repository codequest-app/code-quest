# Worktree Manual System - UI Design

## UI 設計原則

### 1. RPG 沉浸感
- 所有操作都使用 RPG 術語（傳送、時空融合）
- 豐富的圖標和視覺效果
- 動畫和音效增強體驗

### 2. 信息層次
- 關鍵信息突出顯示
- 次要信息適當淡化
- 分組和視覺分隔清晰

### 3. 操作流暢
- 減少操作步驟
- 智能默認值
- 快捷鍵支持

### 4. 狀態可見
- 實時更新
- 清晰的進度指示
- 異常狀態明顯提示

/* 時空裂縫動畫 */

}
