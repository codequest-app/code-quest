**日期**: 2026-02-05

## 核心概念

### 為什麼需要商店系統？

**問題陳述**:
- Claude Code 的功能管理（Skills、MCP、Subagent）過於技術化
- 用戶需要查閱文檔才能了解可用功能
- 缺乏探索和發現的樂趣
- 沒有成長和獎勵的感覺

**商店系統的優勢**:
- ✅ RPG 化功能管理（直觀、有趣）
- ✅ 探索式發現（像逛商店一樣）
- ✅ 成長獎勵系統（解鎖新功能）
- ✅ 引導式學習（新手友好）

### 設計原則

1. **沉浸式體驗**: 所有功能管理都融入 RPG 世界
2. **探索樂趣**: 鼓勵用戶主動探索新功能
3. **成長感**: 隨著等級提升解鎖更強大的功能
4. **簡化複雜**: 用 RPG 概念簡化技術概念
5. **保持實用**: 不只是裝飾，要真正幫助用戶

### 技能升級系統

**升級效果**:
1. 降低 MP 消耗
2. 減少冷卻時間
3. 增加傷害/效果
4. 解鎖新特性

**升級成本**:
```javascript
upgradeCost = basePrice * Math.pow(1.5, currentLevel - 1)

範例:
Lv.1 → Lv.2: 50 金幣
Lv.2 → Lv.3: 75 金幣
Lv.3 → Lv.4: 112 金幣
Lv.4 → Lv.5: 168 金幣
```

## 與現有系統整合

### 場景系統整合

**探索模式**:
```
主菜單:
├─ 📊 狀態
├─ 🔮 技能
├─ 🧙 夥伴
├─ 🏆 成就
├─ 🏪 城鎮 ⭐ 新增
│  ├─ 🔮 技能商店
│  ├─ ⚒️ 工匠鋪
│  ├─ 📚 魔法圖書館
│  ├─ 🧙 傭兵公會
│  ├─ 🏆 寶物庫
│  ├─ 🎯 訓練場
│  └─ 💰 錢莊
├─ 🌲 Worktree
└─ ⚙️ 設定
```

**快捷訪問**:
- 按 `K` 鍵快速打開技能商店
- 按 `G` 鍵快速打開傭兵公會
- 按 `M` 鍵快速打開 MCP 圖書館

### 戰鬥系統整合

**戰鬥中快捷訪問**:
- 技能選擇時顯示技能商店中的資訊
- 召喚夥伴時顯示傭兵公會資訊
- 成就達成時自動彈出寶物庫通知
