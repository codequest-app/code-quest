# 寶物庫 (Treasure Vault)

## 畫面概述

- **功能定位**: 成就、獎勵、收藏品和隱藏內容的展示中心
- **進入條件**: 從商業街進入，或使用快捷鍵 `5` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部標題欄
- 寶物庫名稱和守護者

### 區塊 2: 守護者對話框
- 激勵語

### 區塊 3: 分類標籤
- 成就、獎勵、收藏品、隱藏

### 區塊 4: 成就列表
- 已達成和進行中的成就
- 進度條顯示
