# 商業街 (Shopping District)

## 畫面概述

- **功能定位**: 七大商店的集中入口，商業活動中心
- **進入條件**: 從城鎮廣場進入，或使用快捷鍵 `1` 直接傳送
- **退出條件**: 按 `Esc` 返回城鎮，或進入任一商店
- **場景模式**: 探索模式

## 互動設計

### 鍵盤操作
- **1-7**: 快速進入對應商店
- **Tab**: 循環選擇商店（高亮邊框）
- **Enter**: 進入當前選中的商店
- **Esc**: 返回城鎮廣場
- **方向鍵**: 在商店卡片間移動焦點

### 滑鼠操作
- **點擊商店卡片**: 進入該商店
- **懸停商店卡片**: 顯示懸停效果和簡短描述工具提示
- **右鍵點擊**: 顯示商店詳細資訊彈窗（無需進入）

### 觸控操作（iPad/iPhone）
- **點擊卡片**: 進入商店
- **長按卡片**: 顯示快速預覽（商店概述）
- **滑動**: 滾動查看所有商店（如果螢幕太小）

## 轉場設計

### 進入動畫
- **來源**: 城鎮廣場
- **動畫**:
  - 從右側滑入 (slide-in-right)
  - 商店卡片依序彈出 (stagger animation)
  - 總時長: 600ms
- **時間序列**:
  ```
  0ms    - 背景淡入
  100ms  - 標題滑入
  200ms  - 第 1 排商店彈出（50ms 間隔）
  450ms  - 第 2 排商店彈出（50ms 間隔）
  600ms  - 完成
  ```
- **參考**: `03-flows/screen-transitions.md#shopping-district-enter`
- **音效**: shopping_enter.mp3

### 退出動畫
- **目標**: 城鎮廣場 或 各個商店
- **動畫**:
  - 返回城鎮: 向右滑出 (slide-out-right, 400ms)
  - 進入商店: 商店卡片放大並淡出 (scale + fade, 500ms)
- **音效**:
  - back.mp3 (返回)
  - shop_enter.mp3 (進入商店)

## 狀態變化

### 有新內容
- 商店卡片右上角顯示紅點 🔴
- 例如: 新技能可購買、新成就可領取
- 紅點閃爍動畫（脈動效果）

### 商店暫時關閉
- 卡片變為灰階 (grayscale filter)
- 顯示 "維護中" 文字覆蓋層
- 點擊顯示提示: "商店維護中，請稍後再來"

### 金幣不足提示
- 如果玩家金幣不足以購買任何商品
- 在錢莊卡片上顯示特殊高亮
- 提示: "前往錢莊查看如何獲得更多金幣"

### 首次訪問引導
- 高亮技能商店
- 顯示箭頭和文字: "先去技能商店看看吧！"
- 其他商店卡片暫時半透明

## 無障礙設計

### ARIA 標籤
- `role="navigation"` - 商店列表區域
- `aria-label="商業街商店列表"` - 主容器
- `aria-label="技能商店，按 1 快速進入"` - 各商店按鈕
- `aria-describedby` - 連接到商店描述文字

### 鍵盤導航
- Tab 鍵循環選擇商店
- 焦點高亮明顯（3px 藍色邊框）
- Enter 鍵進入選中商店
- 數字鍵 1-7 快速跳轉

### 螢幕閱讀器
- 播報: "商業街，共有 7 個商店"
- 各商店: "技能商店，按 1 快速進入"
- 新內容提示: "技能商店有新內容可用"

### 高對比度模式
- 商店卡片邊框加粗
- 背景和文字對比度 > 4.5:1
- 焦點指示器使用雙色邊框

## 技術規格

### 效能目標
- 頁面載入: < 500ms
- 卡片懸停響應: < 16ms (60fps)
- 進入商店延遲: < 300ms

### 動畫幀率
- 目標: 60 FPS
- 使用 CSS transform 和 opacity（GPU 加速）
- 避免 layout 重排

### CSS 動畫範例
```css
/* 卡片彈出動畫 */
@keyframes cardPop {
  0% {
    transform: scale(0) rotate(-180deg);
    opacity: 0;
  }
  60% {
    transform: scale(1.1) rotate(10deg);
  }
  100% {
    transform: scale(1) rotate(0deg);
    opacity: 1;
  }
}

/* 懸停效果 */
.shop-card:hover {
  transform: translateY(-4px);
  box-shadow: 0 8px 16px rgba(99, 102, 241, 0.3);
  background: rgba(255, 255, 255, 0.3);
}

/* 新內容紅點脈動 */
@keyframes pulse {
  0%, 100% {
    transform: scale(1);
    opacity: 1;
  }
  50% {
    transform: scale(1.2);
    opacity: 0.7;
  }
}
```
