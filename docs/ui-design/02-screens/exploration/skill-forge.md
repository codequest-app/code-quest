# 工匠鋪 (Skill Forge)

## 畫面概述

- **功能定位**: 自定義技能創建工坊，通過六步驟引導流程鍛造新技能
- **進入條件**: 從商業街進入，或使用快捷鍵 `2` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街，或完成技能創建
- **場景模式**: 探索模式

## 區塊劃分

### 主界面
- **頂部標題欄**: 商店名稱和鍛造師資訊
- **對話區**: 鍛造師的引導語
- **操作選擇**: 兩個主要按鈕（從零開始/使用模板）

### 六步驟創建流程
每個步驟共用以下結構：
- **進度指示器**: 顯示當前步驟（步驟 X/6）
- **主要內容區**: 該步驟的表單或選項
- **底部導航**: [← 上一步] [下一步 →] 或 [取消]

**互動元素**:

- **描述文字區**: 最多 200 字元，即時字數統計

## 互動設計

### 鍵盤操作
- **Tab**: 在表單欄位間移動
- **Enter**: 下一步（如果當前步驟已完成）
- **Esc**: 取消創建（顯示確認對話框）
- **Ctrl+Enter**: 快速保存（在最後一步）
- **←/→**: 上一步/下一步（在進度指示器上）

### 滑鼠操作
- **點擊下一步**: 驗證當前步驟，成功則進入下一步
- **點擊上一步**: 直接返回上一步（不驗證）
- **點擊進度指示器**: 跳轉到對應步驟（已完成的步驟）
- **拖動滑桿**: 調整數值，即時更新顯示

### 觸控操作（iPad/iPhone）
- 所有輸入框自動彈出鍵盤
- 滑桿觸控區域加大（44px）
- 下一步按鈕固定在底部（易於點擊）

## 轉場設計

### 進入動畫
- **來源**: 商業街
- **動畫**: 淡入 (fade-in)
- **時長**: 400ms

### 步驟切換動畫
- **前進**: 當前步驟向左滑出，新步驟從右側滑入
- **後退**: 當前步驟向右滑出，上一步從左側滑入
- **時長**: 300ms

### 完成動畫
- 保存成功後，顯示慶祝動畫（煙火特效）
- 新技能卡片從中心彈出
- 時長: 2s

## 狀態變化

### 表單驗證
- **即時驗證**: 輸入時檢查格式和長度
- **錯誤提示**: 輸入框變紅色，下方顯示錯誤訊息
- **成功提示**: 輸入框變綠色，顯示勾選圖標

### 下一步按鈕狀態
- **禁用**: 當前步驟未完成（灰色）
- **啟用**: 當前步驟已完成（藍色，可點擊）
- **載入中**: 驗證或保存中（旋轉圖標）

### 資料持久化
- 自動保存草稿（每 30 秒）
- 意外關閉時，下次進入可恢復
- 顯示提示: "已自動保存草稿"
