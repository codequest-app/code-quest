# 傭兵公會 (Mercenary Guild / Subagent)

## 畫面概述

- **功能定位**: Subagent 召喚、管理和配置中心
- **進入條件**: 從商業街進入，或使用快捷鍵 `4` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部標題欄
- 公會名稱和會長資訊

### 區塊 2: 會長對話框
- 歡迎語和引導

### 區塊 3: 分類標籤
- 官方傭兵、自定義傭兵、我的隊伍

### 區塊 4: 傭兵列表
- 傭兵卡片展示
- 顯示屬性和專長
