# 訓練場 (Training Ground)

## 畫面概述

- **功能定位**: 新手教學、模擬戰鬥、技能測試和 Combo 練習場所
- **進入條件**: 從商業街進入，或使用快捷鍵 `6` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部標題欄
- 訓練場名稱和訓練師

### 區塊 2: 訓練師對話框
- 激勵語和指導

### 區塊 3: 功能標籤
- 新手教學、模擬戰鬥、技能測試、Combo 練習

### 區塊 4: 課程/訓練列表
- 根據選擇的標籤顯示不同內容
