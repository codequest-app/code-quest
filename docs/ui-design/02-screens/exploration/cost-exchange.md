# 錢莊 (Cost Exchange / Bank)

## 畫面概述

- **功能定位**: 資源管理、AI 模型成本追蹤、預算管理和模型切換
- **進入條件**: 從商業街進入，或使用快捷鍵 `7` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街
- **場景模式**: 探索模式

- 模型成本統計

## 互動設計

### 鍵盤操作
- **Tab**: 在標籤和卡片間循環
- **Enter**: 查看詳情/切換模型
- **Esc**: 返回商業街

### 滑鼠操作
- **點擊 [查看詳情]**: 打開成本詳情彈窗
- **點擊 [切換模型]**: 打開模型選擇對話框
- **點擊圖表**: 顯示該日詳細數據
- **點擊 [導出報表]**: 下載 CSV 報表
