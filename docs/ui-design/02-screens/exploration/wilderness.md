# 野外場景 (Wilderness)

## 畫面概述

- **功能定位**: 探索和隨機遭遇戰的場所，連接城鎮和副本
- **進入條件**: 從城鎮廣場選擇前往野外
- **退出條件**: 按 `B` 返回城鎮，或進入副本
- **場景模式**: 探索模式（可能觸發戰鬥）

## 區塊劃分

### 區塊 1: 頂部狀態欄
- 當前位置: CodeLand - 野外·森林
- 玩家資源: 💰 金幣、⚡ MP

### 區塊 2: 野外地圖視圖（2.5D 俯視）
- 環境元素: 🌲 樹木、⛰️ 山峰、🌋 火山等
- 玩家角色: 👤 可移動圖標
- 副本入口: 🚪 門戶圖標
- 隨機事件點: ❓ 問號圖標

### 區塊 3: 區域資訊欄
- 遭遇戰機率指示（顏色編碼）
- 建議等級範圍

### 區塊 4: 底部操作提示欄
- 鍵盤操作說明
- 快捷鍵列表

## 野外區域類型

### 1. 🌲 森林（Forest）
- **環境**: 樹木密集，綠色基調
- **遭遇戰機率**: 🟡 中等 (30%)
- **敵人類型**: Bug 怪物、語法錯誤精靈
- **建議等級**: Lv. 5-10

### 2. ⛰️ 山脈（Mountains）
- **環境**: 山峰、岩石，灰色基調
- **遭遇戰機率**: 🟠 高 (50%)
- **敵人類型**: 架構巨魔、性能瓶頸獸
- **建議等級**: Lv. 10-15

### 3. 🏜️ 荒野（Wasteland）
- **環境**: 枯木、荒地，金黃色基調
- **遭遇戰機率**: 🔴 極高 (70%)
- **敵人類型**: 技術債惡魔、遺留代碼幽靈
- **建議等級**: Lv. 15-20

### 4. 🌋 火山（Volcano）
- **環境**: 岩漿、煙霧，紅色基調
- **遭遇戰機率**: 🔴🔴 Boss 區 (100%)
- **敵人類型**: 終極 Boss、系統崩潰龍
- **建議等級**: Lv. 20+

## 隨機事件

在野外探索時，偶爾會遇到隨機事件：

```
┌────────────────────────────┐
│   ❓ 隨機事件              │
│                            │
│   你發現了一個寶箱！        │
│                            │
│   選擇:                     │
│   [開啟] - 可能獲得獎勵    │
│   [忽略] - 繼續前進        │
└────────────────────────────┘
```

事件類型：
- 🎁 寶箱（金幣、道具）
- 🧙 神秘商人（特殊商品）
- 📜 任務觸發（額外任務）
- ⚡ 能量恢復點（恢復 MP）

## 互動設計

### 鍵盤操作
- **W/A/S/D**: 移動角色
- **E**: 與副本入口/隨機事件互動
- **B**: 快速返回城鎮
- **I**: 打開物品欄
- **M**: 打開大地圖（顯示當前位置和副本位置）
- **Esc**: 打開選單

### 滑鼠操作
- **左鍵點擊地面**: 角色移動到該位置（尋路）
- **左鍵點擊副本入口**: 顯示副本資訊
- **右鍵**: 快速移動（長距離傳送到點擊位置）

### 觸控操作（iPad/iPhone）
- **點擊地面**: 移動到該位置
- **點擊副本入口**: 顯示資訊
- **虛擬搖桿**: 左下角控制移動

## 轉場設計

### 進入動畫
- **來源**: 城鎮廣場
- **動畫**: 旋轉漸變 (800ms)
- **音效**: scene_change.mp3
- **視覺**: 場景從城鎮淡出，野外淡入

### 退出動畫
- **目標**: 城鎮廣場 或 副本
- **動畫**:
  - 返回城鎮: 旋轉漸變 (800ms)
  - 進入副本: 門戶吸入效果 (1000ms)
- **音效**:
  - scene_change.mp3 (返回)
  - portal_enter.mp3 (進入副本)

### 戰鬥觸發轉場
- 見「遭遇戰觸發」章節
- 震動 + 閃紅 + 敵人出現動畫
- 總時長: 2.7 秒

## 技術規格

### 遭遇戰計算
```javascript
function checkRandomEncounter(playerLevel, areaType) {
  const baseRate = areaType.encounterRate // 0.30 for forest
  const levelModifier = (playerLevel - areaType.recommendedLevel) * 0.05
  const finalRate = Math.max(0.1, Math.min(0.9, baseRate + levelModifier))

  return Math.random() < finalRate
}
```

### 移動步數追蹤
- 每移動 5-10 步檢查一次遭遇戰
- 累積步數獎勵（探索獎勵）
