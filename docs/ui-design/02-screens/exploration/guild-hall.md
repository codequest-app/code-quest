# 公會大廳 (Guild Hall / Worktree Manager)

## 畫面概述

- **功能定位**: 時空管理器，管理所有 Git Worktrees（主世界、戰鬥分支、手動分支）
- **進入條件**: 從城鎮廣場進入，或使用快捷鍵 `G` 直接傳送
- **退出條件**: 按 `Esc` 返回城鎮廣場
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部控制欄
- 時空管理器標題
- 創建新時間線按鈕
- 視圖切換按鈕（列表/時間線圖/看板）
- 排序選項下拉選單

### 區塊 2: 主世界卡片
- 主分支 (main) 的狀態展示
- 受保護標示
- 查看詳情按鈕

### 區塊 3: 時間線分類列表
- 按類型分組: ⚔️ 冒險世界、🛡️ 修復世界、🔮 實驗世界
- 每個分類可展開/摺疊
- 顯示活躍數量

### 區塊 4: 時間線卡片
- 完整的 worktree 資訊展示
- 進度條和里程碑
- 操作按鈕

### 區塊 5: 右側統計面板（可選，桌面版）

## 互動設計

### 鍵盤操作
- **Ctrl+N**: 創建新時間線
- **Ctrl+1-9**: 切換到時間線 #1-9
- **Ctrl+0**: 切換到主世界
- **Ctrl+W**: 關閉當前時間線
- **Ctrl+M**: 合併當前時間線
- **Ctrl+S**: 暫存變更
- **Tab**: 在時間線卡片間循環
- **Enter**: 進入選中的時間線
- **Esc**: 返回城鎮廣場

### 滑鼠操作
- **點擊 [進入]**: 切換到該時間線
- **點擊 [合併]**: 打開合併對話框
- **點擊 [關閉]**: 顯示關閉確認（如有未保存變更）
- **點擊視圖按鈕**: 切換視圖模式
- **拖拽卡片**: （看板模式）改變狀態
