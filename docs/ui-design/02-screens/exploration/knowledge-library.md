# 魔法圖書館 (Knowledge Library / MCP Tools)

## 畫面概述

- **功能定位**: MCP 工具的安裝、管理和配置中心
- **進入條件**: 從商業街進入，或使用快捷鍵 `3` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部標題欄
- 圖書館名稱和館長資訊
- 返回按鈕

### 區塊 2: 館長對話框
- 歡迎語或使用提示

### 區塊 3: 篩選和搜尋欄
- 標籤篩選：已安裝、熱門、最新、分類
- 搜尋框：即時搜尋 MCP 工具

### 區塊 4: MCP 工具列表
- 工具卡片垂直排列
- 已安裝和未安裝視覺區分
- 可滾動查看更多

## 互動設計

### 鍵盤操作
- **Tab**: 在篩選標籤和卡片間循環
- **Enter**: 執行主要操作（安裝/配置）
- **Esc**: 返回商業街
- **↑/↓**: 在卡片間移動
- **Ctrl+F**: 聚焦到搜尋框

### 滑鼠操作
- **點擊篩選標籤**: 切換篩選
- **點擊 [安裝]**: 開始安裝流程
- **點擊 [配置]**: 打開配置對話框
- **點擊 [卸載]**: 顯示確認對話框
- **點擊工具名稱**: 查看詳情
