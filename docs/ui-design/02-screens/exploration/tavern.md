# 酒館 (Tavern)

## 畫面概述

- **功能定位**: AI 對話中心、任務公告欄、探索模式的主要互動場所
- **進入條件**: 從城鎮廣場進入，或使用快捷鍵 `H` 直接傳送
- **退出條件**: 按 `Esc` 返回城鎮廣場
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部標題欄
- 酒館名稱: The Code Tavern
- 返回按鈕
- 背景色: 溫暖的木質色調 (#3e2723)

### 區塊 2: 場景視覺區
- 酒館老闆 NPC（AI 化身）
- 蠟燭裝飾 🕯️
- 對話氣泡（動態文字）
- 開始對話按鈕

### 區塊 3: 對話區（主要互動區）
- 對話輸入框（多行文字區）
- 發送和清空按鈕
- 快速問題按鈕列表

### 區塊 4: 對話歷史區（可選，摺疊）
- 顯示最近的 5-10 條對話
- 可滾動查看歷史
- 點擊重新提問

- 多行文字區，支援 Markdown
- 字數統計（最多 2000 字元）
- 發送按鈕（Enter 快捷鍵）

- 用戶和 AI 的對話視覺區分

- AI: 左側，灰色背景
- Markdown 渲染（代碼高亮）
- 複製按鈕、重新提問按鈕

## 互動設計

### 鍵盤操作
- **Enter**: 發送訊息（Shift+Enter 換行）
- **Esc**: 返回城鎮廣場
- **↑**: 填入上一次的問題（命令行風格）
- **Ctrl+L**: 清空對話歷史
- **Ctrl+K**: 聚焦到輸入框

### 滑鼠操作
- **點擊 [開始對話]**: 展開對話輸入區
- **點擊 [發送]**: 提交問題，顯示 AI 回答
- **點擊快速問題**: 自動填入輸入框並發送
- **點擊 [💾 保存]**: 保存對話到歷史
- **點擊 [🔄 重新提問]**: 重新發送同樣的問題
- **懸停對話氣泡**: 顯示操作按鈕（複製、保存等）

### 觸控操作（iPad/iPhone）
- **點擊輸入框**: 彈出鍵盤
- **長按對話氣泡**: 顯示快速操作選單（複製/保存/分享）
- **滑動對話**: 滾動查看歷史

## 轉場設計

### 進入動畫
- **來源**: 城鎮廣場
- **動畫**: 推門進入效果（門從中間打開）
- **時長**: 800ms
- **音效**: door_open.mp3, tavern_ambience.mp3（背景音樂）

### 退出動畫
- **目標**: 城鎮廣場
- **動畫**: 門關閉效果
- **時長**: 600ms
- **音效**: door_close.mp3

## 狀態變化

### 等待 AI 回應
- 酒館老闆顯示「思考中」動畫（...）
- 對話氣泡顯示載入指示器
- 輸入框禁用

### AI 回應中（流式輸出）
- 打字效果（逐字顯示）
- 用戶可以隨時中斷（停止按鈕）

### 對話歷史已滿
- 提示: "對話歷史已達上限，舊對話將被清除"
- 提供下載對話歷史的選項

### 首次訪問
- 酒館老闆主動打招呼
- 顯示教學提示: "在這裡你可以向 AI 提問，不會觸發戰鬥"

  messages: [
    {
      role: 'user',

    },
    {
      role: 'assistant',
      content: '以下是優化 React 性能的幾個方法...',
