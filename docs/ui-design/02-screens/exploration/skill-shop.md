# 技能商店 (Skills Shop)

## 畫面概述

- **功能定位**: 購買、升級和管理官方及自定義技能
- **進入條件**: 從商業街進入，或使用快捷鍵 `1` 直接傳送
- **退出條件**: 按 `Esc` 返回商業街
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部標題欄
- 商店名稱和圖標
- 店主資訊
- 返回按鈕

### 區塊 2: 店主對話框
- 歡迎語或提示訊息
- 根據玩家狀態動態變化

### 區塊 3: 篩選標籤欄
- 4 個標籤: 官方技能、自定義技能、已解鎖、未解鎖
- 可組合篩選（例如: 官方 + 未解鎖）

### 區塊 4: 技能列表（主要內容，可滾動）
- 技能卡片垂直排列
- 每個卡片顯示完整技能資訊
- 已解鎖和未解鎖視覺區分

### 區塊 5: 底部分頁控制
- 上一頁/下一頁按鈕
- 頁碼指示

## 互動設計

### 鍵盤操作
- **Tab**: 在篩選標籤和技能卡片間循環
- **Enter**: 對焦點元素執行主要操作（使用/解鎖/升級）
- **Esc**: 返回商業街
- **↑/↓**: 在技能卡片間移動焦點
- **Space**: 查看詳情
- **PgUp/PgDn**: 翻頁

### 滑鼠操作
- **點擊篩選標籤**: 切換篩選條件
- **點擊 [使用]**: 立即使用該技能（如果不在戰鬥中，顯示提示）
- **點擊 [查看詳情]**: 打開技能詳情彈窗
- **點擊 [升級]**: 打開升級確認對話框
- **點擊 [解鎖]**: 打開解鎖確認對話框（檢查條件）
- **懸停卡片**: 輕微高亮邊框
- **右鍵點擊**: 顯示快速操作選單

### 觸控操作（iPad/iPhone）
- **點擊卡片**: 展開顯示完整資訊（如果是摺疊狀態）
- **長按卡片**: 顯示快速操作選單
- **滑動卡片**: 顯示隱藏的操作按鈕（左滑：升級，右滑：使用）
- **下拉刷新**: 重新載入技能列表

## 轉場設計

### 進入動畫
- **來源**: 商業街
- **動畫**:
  - 從下方滑入 (slide-up)
  - 技能卡片依序淡入 (stagger fade-in, 50ms 間隔)
- **時長**: 600ms
- **音效**: shop_open.mp3

### 退出動畫
- **目標**: 商業街
- **動畫**: 向下滑出 (slide-down)
- **時長**: 400ms
- **音效**: shop_close.mp3

### 技能詳情彈窗
- **進入**: 縮放淡入 (scale + fade-in, 300ms)
- **退出**: 縮放淡出 (scale + fade-out, 200ms)

## 狀態變化

### 技能升級成功
- 技能卡片閃爍金色光芒
- 等級數字彈出動畫
- 新屬性數值高亮顯示（綠色）
- 顯示提示: "升級成功！MP 消耗降低"

### 技能解鎖成功
- 卡片從灰色變為彩色（漸變動畫）
- 🔒 圖標變為 ✅（旋轉切換）
- 煙火特效
- 顯示提示: "恭喜解鎖新技能！"

### 金幣/等級不足
- 嘗試解鎖/升級時
- 按鈕震動動畫
- 顯示紅色提示: "金幣不足" 或 "等級不足"

### 篩選條件變化
- 技能卡片淡出舊列表
- 淡入新列表（stagger animation）
- 顯示篩選結果數量: "找到 8 個技能"

### 首次訪問
- 高亮第一個技能卡片
- 顯示教學提示: "這是你的第一個技能，點擊查看詳情"

## 響應式設計

### 桌面 (1024px+)
- 每行顯示 1 個完整技能卡片
- 側邊欄顯示快速統計（已解鎖技能數量等）
- 詳情彈窗居中顯示，寬度 600px

### 平板 (768-1023px)
- 技能卡片寬度自適應
- 詳情彈窗寬度 80%
- 篩選標籤可橫向滾動

### 手機 (<768px)
- 技能卡片摺疊顯示（僅標題和圖標）
- 點擊展開顯示詳細資訊
- 詳情彈窗全屏顯示
- 篩選標籤改為下拉選單

## 無障礙設計

### ARIA 標籤
- `role="list"` - 技能列表
- `role="listitem"` - 各技能卡片
- `aria-label="版本封印術，等級 3，已解鎖"` - 技能卡片
- `aria-expanded` - 展開/摺疊狀態
- `aria-disabled` - 不可用的按鈕

### 鍵盤導航
- 完整鍵盤操作支援
- 焦點指示器明顯（3px 藍色邊框）
- Tab 順序邏輯清晰

### 螢幕閱讀器
- 播報技能完整資訊
- 解鎖條件詳細說明
- 操作結果反饋

## 技術規格

### 效能目標
- 技能列表載入: < 300ms
- 卡片渲染: < 16ms per card
- 篩選響應: < 100ms
- 記憶體使用: < 50MB

### 虛擬滾動
- 使用虛擬滾動優化長列表（> 50 項）
- 僅渲染可見區域 + 緩衝區
- 滾動時動態載入/卸載
