# 城鎮廣場 (Town Square)

## 畫面概述

- **功能定位**: 探索模式的主場景，作為所有商店和設施的入口中心
- **進入條件**: 從野外/副本返回，或使用快捷鍵 `B` 返回城鎮
- **退出條件**: 進入任何商店/建築，或前往野外區域
- **場景模式**: 探索模式

## 區塊劃分

### 區塊 1: 頂部狀態欄
- 當前位置名稱: CodeLand - 城鎮區域
- 玩家資源顯示: 💰 金幣、⚡ MP
- 固定在頂部，始終可見

### 區塊 2: 地圖主視圖（2.5D 俯視）
- 建築物圖標和名稱標示
- 玩家角色 👤 可移動圖標
- 可互動建築高亮顯示
- NPC 角色位置標示

### 區塊 3: 底部操作提示欄
- 鍵盤操作說明
- 快捷鍵列表
- 動態更新（靠近建築時顯示互動提示）

## 組件清單

### 共用組件（引用）
- **狀態欄**: `04-components/status-displays.md` - 頂部資源顯示
- **操作提示**: `04-components/input-hints.md` - 底部提示欄
- **地圖圖標**: `04-components/icons.md` - 建築物圖標

### 特有組件
- **建築入口**: 可點擊的建築物區域，顯示名稱和圖標
  - 懸停時: 邊框高亮，顯示 "按 E 進入" 提示
  - 距離檢測: 玩家角色靠近時自動顯示互動提示

- **玩家角色**: 👤 可移動的角色圖標
  - WASD 控制移動
  - 移動動畫（輕微位移）
  - 碰撞檢測

- **NPC 角色**: 🧙‍♂️ 固定位置的 NPC
  - 可互動標示
  - 對話氣泡提示

## 互動設計

### 鍵盤操作
- **W/↑**: 向上移動角色
- **A/←**: 向左移動角色
- **S/↓**: 向下移動角色
- **D/→**: 向右移動角色
- **E/Enter**: 進入當前靠近的建築
- **Esc**: 打開系統選單
- **M**: 打開大地圖
- **Tab**: 快速場所選單
- **I**: 物品欄
- **C**: 角色面板
- **H**: 快速傳送到酒館
- **G**: 快速傳送到公會大廳
- **P**: 進入靜止之間
- **1-7**: 快速傳送到對應商店
- **B**: 返回城鎮（從其他區域）

### 滑鼠操作
- **左鍵點擊建築**: 進入建築
- **左鍵點擊地面**: 角色移動到該位置（尋路）
- **右鍵點擊建築**: 顯示建築資訊彈窗
- **滾輪**: 縮放地圖（保持在最小/最大限制內）

### 觸控操作（iPad/iPhone）
- **點擊建築**: 進入建築
- **點擊地面**: 角色移動到該位置
- **長按建築**: 顯示建築資訊
- **拖曳玩家圖標**: 實時移動角色（跟隨手指）
- **雙指縮放**: 縮放地圖
- **雙指拖曳**: 移動視角

### 虛擬按鈕（移動端）
```
左下角:
┌─────────┐
│  ↑      │
│ ← ↓ →   │  方向搖桿
└─────────┘

右下角:
┌─────────┐
│   [E]   │  互動按鈕
│ 進入建築 │
└─────────┘

右上角:
┌─────────┐
│ [M] [I] │  地圖/物品
│ [Tab]   │  選單
└─────────┘
```

## 轉場設計

### 進入動畫
- **來源**: 野外場景、副本出口、其他商店
- **動畫**: 漸變淡入 (fade-in)
- **時長**: 500ms
- **參考**: `03-flows/screen-transitions.md`
- **音效**: town_enter.mp3

### 退出動畫
- **目標**: 商業街、酒館、公會大廳、野外、靜止之間
- **動畫**:
  - 進入建築: 漸變淡出 + 門開啟效果 (500ms)
  - 快捷鍵傳送: 閃光效果 (300ms)
  - 前往野外: 旋轉漸變 (800ms)
- **音效**:
  - door_open.mp3 (進入建築)
  - teleport.mp3 (快捷鍵)
  - scene_change.mp3 (前往野外)

## 狀態變化

### 初次進入（新玩家）
- 顯示教學提示覆蓋層
- 高亮商業街和酒館
- 引導文字: "歡迎來到 CodeLand！先去商業街看看吧"

### 有未完成任務
- 相關建築物上方顯示 ❗ 圖標
- 閃爍提示吸引注意
- 底部提示: "你有未完成的任務"

### 低資源警告
- MP < 20: 玩家住所高亮，提示 "MP 不足，去休息恢復"
- 金幣 < 50: 提示 "金幣不足，完成任務獲得獎勵"

### 新內容可用
- 建築物上方顯示 🆕 或紅點
- 例如: 新技能可購買、新夥伴可召喚

## 響應式設計

### 桌面 (1024px+)
- 完整地圖視圖，所有建築可見
- 側邊欄顯示快速狀態和小地圖
- 懸停效果和工具提示

### 平板 (768-1023px)
- 地圖佔滿全屏
- 快速狀態浮動在右上角
- 虛擬控制器顯示在螢幕底部

### 手機 (<768px)
- 地圖縮放以適應螢幕
- 建築物名稱在點擊時顯示
- 虛擬搖桿和按鈕較大，易於點擊
- 雙擊建築快速進入

## 無障礙設計

### ARIA 標籤
- `role="navigation"` - 主地圖區域
- `aria-label="城鎮地圖"` - 地圖容器
- `aria-label="商業街入口"` - 各建築按鈕
- `aria-live="polite"` - 動態提示區域

### 鍵盤導航
- Tab 鍵循環選擇建築
- Enter 進入選中的建築
- Shift+Tab 反向選擇
- 焦點高亮明顯可見

### 螢幕閱讀器
- 建築物名稱語音播報
- 當前位置描述: "城鎮廣場，共有 6 個可進入的建築"
- 操作結果反饋: "已進入商業街"

### 色盲模式
- 高對比度選項
- 不僅依賴顏色區分建築，也使用圖標和形狀
- 焦點邊框使用多種樣式（虛線、實線、加粗）

## 技術規格

### 效能目標
- 初次載入時間: < 2 秒
- 場景切換: < 500ms
- 角色移動響應: < 16ms (60fps)
- 記憶體使用: < 100MB

### 動畫幀率
- 目標: 60 FPS
- 角色移動: 平滑插值
- 建築高亮: CSS transition
- 場景切換: requestAnimationFrame

### 資源載入策略
- **預載入**: 當前城鎮的所有建築圖標和音效
- **延遲載入**: 副本和野外場景資源
- **快取**: 常用場景數據存儲在 localStorage
- **優化**: 圖片使用 WebP 格式，音效使用壓縮格式
