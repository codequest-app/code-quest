### 問題 #8: 傷害計算

**驗證點**:
- [ ] baseDamage = 100 + (MP×3) + (level×10)
- [ ] 親和性倍率正確應用
- [ ] 弱點倍率 = 1.5
- [ ] 抗性倍率 = 0.5
- [ ] 向下取整正確

**測試方法**:
```
測試用例（Lv.10 玩家，15 MP 技能）:
1. baseDamage = 100 + 45 + 100 = 245
2. 水屬性 vs 火敵人 = 245 × 1.5 = 367.5
3. 最終傷害 = 368（向下取整）
```
