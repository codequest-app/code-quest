# Worktree 工作流程

**日期**: 2026-02-05
**版本**: v1.0

---

## 流程概述

本文檔詳細說明 Worktree 管理系統，從公會大廳導航、視圖模式切換、創建新分支、切換世界、合併分支到清理歸檔的完整流程，以及與非同步戰鬥的整合。

### 目的
- 提供完整的 Worktree 管理指引
- 支援多任務並行處理
- 整合非同步戰鬥系統
- 確保分支操作的安全性

### 入口點
- 快捷鍵 G 前往公會大廳
- 城鎮地圖點擊公會建築
- 非同步戰鬥自動創建

### 預期結果
- 成功管理多個工作世界
- 安全合併分支內容
- 並行執行多個戰鬥任務

### 相關流程
- 【參考：async-battle-flow.md】- 非同步戰鬥
- 【參考：battle-flow.md】- 戰鬥系統
- 【參考：screen-transitions.md】- 場景轉場

## 階段 2：視圖模式切換

### 2.1 列表視圖（預設）

```
[列表視圖特性]
├─→ 以卡片形式顯示所有 Worktree
├─→ 顯示狀態圖示和簡要資訊
├─→ 支援快速操作按鈕
└─→ 適合快速瀏覽和操作
```

### 2.2 時間線視圖

```
[時間線視圖特性]
├─→ 按時間軸排列 Worktree
├─→ 顯示創建、更新、合併時間點
├─→ 可視化分支演進過程
└─→ 適合了解歷史變化

時間線示例：
───────────────────────────────────────→ 時間
  │         │         │         │
  ▼         ▼         ▼         ▼
main    創建分支1  創建分支2  合併分支1
2/1       2/2       2/4       2/5
```

### 2.3 看板視圖

```
[看板視圖特性]
├─→ 按狀態分組（待開始/進行中/已完成）
├─→ 拖放排序和移動
├─→ 視覺化工作流程
└─→ 適合項目管理場景

看板布局：
┌──────────┬──────────┬──────────┐
│ 待開始   │ 進行中   │ 已完成   │
├──────────┼──────────┼──────────┤
│ 分支 A   │ 分支 B   │ 分支 C   │
│          │ ⚔ 戰鬥中 │ ✓ 已合併 │
└──────────┴──────────┴──────────┘
```

## 階段 4：切換 Worktree

```
[步驟 1] 選擇要切換的 Worktree
           │
           ▼
[步驟 2] 點擊「切換」按鈕
           │
           ▼
[步驟 3] <檢查切換條件>
           │
           ├─→ 當前世界是否有未保存更改？
           ├─→ 目標世界是否正在戰鬥中？
           └─→ 是否有衝突需要解決？
           │
           ▼
[步驟 4] <條件檢查結果>
           │  ╱      ╲
           │ 通過    未通過
           │ │       │
           │ │       ▼
           │ │    顯示錯誤/警告
           │ │    - 請先保存更改
           │ │    - 無法切換至戰鬥中的世界
           │ │    - 需要先解決衝突
           │ │       │
           │ ▼       ↻
[步驟 5] 顯示確認對話框
           │
           │ "切換至「冒險世界-2」？"
           │ "當前進度將被保存"
           │ [確認] [取消]
           │    │      │
           │    │      ▼
           │    │   返回列表
           │    │
           │    ▼
[步驟 6] 執行切換操作
           │
           ├─→ 保存當前世界狀態
           ├─→ Git worktree checkout
           ├─→ 載入目標世界狀態
           └─→ 更新 UI
           │
           ▼
[步驟 7] 播放轉場動畫
           │
           ├─→ 畫面淡出 (0.3s)
           ├─→ 切換場景
           └─→ 畫面淡入 (0.3s)

           ▼

           │
           │ "✓ 已切換至「冒險世界-2」"
           │
           ▼
     返回探索模式（新世界）
```

## Worktree 狀態管理

### 狀態類型

| 狀態 | 圖示 | 說明 | 可執行操作 |
|-----|------|------|----------|
| 主世界 | 🏠 | main 分支 | 查看、無法刪除 |
| 當前 | ✓ | 正在使用 | 查看、合併、刪除 |
| 閒置 | ⏸ | 未使用 | 切換、合併、刪除 |
| 戰鬥中 | ⚔ | 非同步戰鬥 | 查看進度 |
| 待合併 | ⬆ | 等待合併 | 合併、刪除 |
| 已歸檔 | 📦 | 已歸檔 | 查看、還原 |

### 狀態轉換

```
創建 → 閒置 → 當前 → 待合併 → 已合併/刪除
        ↓      ↓
      戰鬥中  歸檔
```

## 用戶體驗考量

### 視覺回饋

**切換動畫**:
- 淡出淡入 (0.6s 總時長)
- 進度指示器
- 平滑過渡

**合併進度**:
- 進度條顯示
- 步驟指示
- 預計時間

### 錯誤處理

**常見錯誤**:
1. 切換至戰鬥中的世界
   - 提示：該世界正在戰鬥中，無法切換
   - 解決：等待戰鬥結束

2. 合併衝突
   - 提示：檢測到衝突，需要手動解決
   - 解決：選擇保留哪個版本

3. 刪除主世界
   - 提示：主世界無法刪除
   - 解決：只能刪除其他分支

### 快捷鍵

- `G` - 打開公會大廳
- `N` - 創建新 Worktree
- `1-9` - 快速切換至指定 Worktree
- `M` - 合併當前 Worktree
- `Esc` - 返回探索模式
