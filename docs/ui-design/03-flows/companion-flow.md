# 夥伴召喚和管理流程

**日期**: 2026-02-05
**版本**: v1.0

---

## 流程概述

本文檔詳細說明夥伴系統的完整流程，從傭兵公會訪問、瀏覽可用夥伴、召喚新夥伴、管理夥伴陣型、戰鬥中的自動行動，到夥伴升級和培養。

### 目的
- 提供完整的夥伴系統指引
- 說明召喚和管理機制
- 整合戰鬥系統
- 支援陣型策略配置

### 入口點
- 傭兵公會（商業街）
- 角色管理畫面
- 戰鬥準備階段

### 預期結果
- 成功召喚和管理夥伴
- 配置最佳戰鬥陣型
- 提升整體戰鬥能力

### 相關流程
- 【參考：battle-flow.md】- 戰鬥中夥伴行動
- 【參考：shop-flow.md】- 商店系統
- 【參考：screen-transitions.md】- 場景轉場

## 階段 3：召喚新夥伴

```
[步驟 1] 點擊「召喚」按鈕
           │
           ▼
[步驟 2] 顯示召喚確認對話框
           │
           ├─→ 夥伴資訊
           ├─→ 召喚成本
           ├─→ 槽位檢查
           └─→ 金幣檢查
           │
           ▼
[步驟 3] <檢查召喚條件>
           │
           ├─→ 槽位是否已滿？
           ├─→ 金幣是否足夠？
           └─→ 等級是否符合？
           │
           ▼
[步驟 4] <條件檢查結果>
           │  ╱      ╲
           │ 通過    未通過
           │ │       │
           │ │       ▼
           │ │    顯示錯誤
           │ │    - 槽位已滿，請先解散
           │ │    - 金幣不足 (需要 XXX)
           │ │    - 等級不足 (需要 Lv.X)
           │ │       │
           │ ▼       ↻
[步驟 5] 用戶確認
           │
           │ "確認召喚？"
           │ [確認] [取消]
           │    │      │
           │    │      ▼
           │    │   返回列表
           │    │
           │    ▼
[步驟 6] 扣除金幣
           │
           │ Gold -= cost
           │
           ▼
[步驟 7] 播放召喚動畫
           │
           ├─→ 魔法陣出現 (0.5s)
           ├─→ 光芒閃爍 (1.0s)
           ├─→ 夥伴登場 (0.5s)
           └─→ 音效：summon.wav
           │
           ▼
[步驟 8] 夥伴加入隊伍
           │
           ├─→ 生成夥伴數據
           ├─→ 加入夥伴列表
           ├─→ 分配預設陣型
           └─→ 記錄召喚時間
           │
           ▼
[步驟 9] 顯示成功提示
           │
           │ "✓ 成功召喚「戰士・艾倫」！"
           │
           ▼
[步驟 10] <立即配置陣型?>
           │  ╱      ╲
           │ 是       否
           │ │        │
           │ ▼        ▼
    進入陣型配置  返回公會主界面
```

### 夥伴 AI 邏輯

```javascript
function companionAI(companion, battleState) {
  const playerHpPercent = battleState.player.hp / battleState.player.maxHp;
  const enemyHpPercent = battleState.enemy.hp / battleState.enemy.maxHp;

  // 決策樹
  if (companion.type === "tank") {
    // 坦克型：優先保護
    if (playerHpPercent < 0.3) {
      return useTauntSkill(companion); // 嘲諷吸引攻擊
    } else {
      return useBasicAttack(companion);
    }
  } else if (companion.type === "dps") {
    // 輸出型：優先傷害
    if (enemyHpPercent < 0.2) {
      return useFinisherSkill(companion); // 終結技
    } else {
      return useStrongSkill(companion);
    }
  } else if (companion.type === "healer") {
    // 治療型：優先治療
    if (playerHpPercent < 0.5) {
      return useHealSkill(companion, battleState.player);
    } else {
      return useBuffSkill(companion);
    }
  }
}
```
