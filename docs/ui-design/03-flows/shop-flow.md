# 商店購物流程

**日期**: 2026-02-05
**版本**: v1.0

---

## 流程概述

本文檔說明商店系統流程，從商業街導航、7 種商店瀏覽、物品/技能選購、購買確認到庫存更新的完整流程。

### 目的
- 提供完整的購物體驗
- 說明 7 種商店的特性
- 規範購買和交易流程

### 入口點
- 快捷鍵 1-7 直達商店
- 商業街點擊商店圖標
- 任務導引

### 預期結果
- 成功購買物品/技能
- 自動更新庫存
- 扣除對應金幣

### 相關流程
- 【參考：skill-creation-flow.md】- 技能系統
- 【參考：screen-transitions.md】- 場景轉場

## 階段 1：7 種商店介紹

### 1. 技能商店 (快捷鍵 1)

```
商品類型：預設技能
價格範圍：200-5000 Gold
特色：即買即用，無需配置

商品分類：
├─ 攻擊技能 (15 種)
├─ 防禦技能 (8 種)
├─ 輔助技能 (12 種)
└─ 召喚技能 (6 種)
```

### 2. 工匠鋪 (快捷鍵 2)

```
商品類型：裝備和武器
價格範圍：100-3000 Gold
特色：提升屬性

商品分類：
├─ 武器 (劍、弓、法杖)
├─ 防具 (盔甲、盾牌)
├─ 飾品 (戒指、項鍊)
└─ 材料 (鍛造素材)
```

### 3. 圖書館 (快捷鍵 3)

```
商品類型：知識和情報
價格範圍：50-1000 Gold
特色：解鎖資訊、提升屬性

商品分類：
├─ 技能書 (學習新技能)
├─ 地圖 (解鎖區域)
├─ 怪物圖鑑
└─ 攻略本
```

### 4. 傭兵公會 (快捷鍵 4)

```
商品類型：夥伴召喚
價格範圍：500-5000 Gold
特色：戰鬥助力

參考：【參考：companion-flow.md】
```

### 5. 寶物庫 (快捷鍵 5)

```
商品類型：稀有道具
價格範圍：1000-10000 Gold
特色：珍稀物品、限量供應

商品分類：
├─ 消耗品 (藥水、卷軸)
├─ 特殊道具 (復活石、傳送石)
├─ 收藏品 (裝飾、成就)
└─ 抽獎券
```

### 6. 訓練場 (快捷鍵 6)

```
商品類型：訓練服務
價格範圍：免費-500 Gold/次
特色：技能測試、屬性提升

服務項目：
├─ 技能測試 (免費)
├─ 特訓課程 (+EXP)
├─ 對戰模擬
└─ 屬性重置
```

### 7. 錢莊 (快捷鍵 7)

```
商品類型：金融服務
特色：存款、貸款、兌換

服務項目：
├─ 存款 (利息 2%/天)
├─ 貸款 (利息 5%/天)
├─ 貨幣兌換
└─ 交易記錄
```

           ▼

## 特殊購買流程

### 抽獎/隨機商品

```
[步驟 1] 選擇抽獎商品
         │
         ▼
[步驟 2] 支付抽獎券或金幣
         │
         ▼
[步驟 3] 播放抽獎動畫
         │
         ├─→ 轉盤旋轉 (2s)
         ├─→ 逐漸減速
         └─→ 停在獎品上
         │
         ▼
[步驟 4] 顯示獲得的獎品
         │
         ├─→ 稀有度展示
         ├─→ 獎品資訊
         └─→ 加入庫存
         │
         ▼
[步驟 5] <再抽一次?>
         │  ╱      ╲
         │ 是       否
         │ │        │
         │ ↻        ▼
      重複抽獎   返回商店
```

### 批量購買

```
[步驟 1] 選擇商品
         │
         ▼
[步驟 2] 點擊「批量購買」
         │
         ▼
[步驟 3] 選擇數量
         │
         │ 滑桿：1 - 99
         │
         ▼
[步驟 4] 計算總價
         │
         │ 總價 = 單價 × 數量 × 折扣
         │
         ▼
[步驟 5] 確認購買
         │
         ▼
     批量加入庫存
```

## 商店特殊機制

### 每日特價

```
- 每日更新 3 個特價商品
- 折扣 10-30%
- 限量供應
- 重置時間：每日 00:00
```

### 會員等級折扣

| 等級 | 累計消費 | 折扣 | 特權 |
|-----|---------|------|------|
| 普通 | 0 | 0% | - |
| 銅牌 | 1,000 | 5% | 優先通知 |
| 銀牌 | 5,000 | 10% | 專屬商品 |
| 金牌 | 20,000 | 15% | 定製服務 |
| 鑽石 | 100,000 | 20% | VIP 待遇 |

```
活動類型：
├─ 節日促銷（全場 8 折）

活動提示：

推送通知
```

## 用戶體驗考量

### 視覺回饋
- 購買成功動畫
- 金幣扣除動畫
- 商品加入庫存動畫
- 音效反饋

### 錯誤處理
- 金幣不足：提示最近賺錢方式
- 庫存已滿：提示整理背包
- 等級不足：顯示升級進度

### 快捷操作
- `1-7` 快速切換商店
- `Enter` 購買
- `Esc` 返回
- `/` 搜尋
