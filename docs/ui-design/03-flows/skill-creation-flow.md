# 技能創建流程

**日期**: 2026-02-05
**版本**: v1.0

---

## 流程概述

本文檔詳細說明技能創建系統的完整流程，從進入技能鍛造所、選擇基礎技能模板、調整 Prompt、配置工具權限、設定元數據，到測試和保存的完整 6 步驟流程。

### 目的
- 提供完整的技能創建指引
- 說明每個創建步驟的細節
- 整合模板庫和測試系統
- 確保創建的技能符合需求

### 入口點
- 從城鎮前往技能鍛造所
- 快捷鍵直達（數字鍵）
- 技能管理畫面的「創建新技能」按鈕

### 預期結果
- 成功創建客製化技能
- 技能自動加入技能列表
- 可在戰鬥中立即使用

- 【參考：battle-flow.md】- 戰鬥中使用技能
- 【參考：shop-flow.md】- 購買預設技能
- 【參考：screen-transitions.md】- 場景轉場

### Prompt 最佳實踐

**✓ 好的 Prompt**:
```
"釋放冰霜魔法攻擊 {target}，造成冰屬性傷害並使其減速 2 回合"
```
- 清晰的動作描述
- 包含必要變數
- 說明效果和持續時間

**✗ 不好的 Prompt**:
```
"攻擊敵人"
```
- 太模糊
- 缺少變數
- 沒有說明效果

**✗ 過度複雜的 Prompt**:
```
"召喚遠古冰霜之神的力量，透過次元裂縫引導無盡的冰雪風暴，席捲整個戰場，凍結所有敵人的靈魂..."
```
- 過於冗長
- 描述過度
- AI 難以準確解讀
