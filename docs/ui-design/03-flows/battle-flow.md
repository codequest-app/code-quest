# 戰鬥流程詳解

**日期**: 2026-02-05
**版本**: v1.0

---

## 流程概述

本文檔詳細說明戰鬥系統的完整流程，從戰鬥觸發、準備階段、回合制戰鬥循環、技能執行、夥伴和召喚獸整合，到戰鬥結束後的獎勵和升級系統。

### 目的
- 定義完整的戰鬥機制和規則
- 說明回合制戰鬥的邏輯流程
- 整合夥伴和召喚獸系統
- 規範獎勵計算和分配

### 入口點
- 任務型 Prompt（自動觸發）
- 隨機遭遇戰
- 副本入口主動挑戰
- NPC 任務觸發

### 預期結果
- 戰鬥勝利：獲得 EXP、Gold、物品掉落
- 戰鬥失敗：扣除 MP、HP 保留
- 逃跑成功：扣除 30% MP
- 角色升級：屬性提升、技能解鎖

```

           │
           ▼
```

```
           │
           ▼
[步驟 3] <判定為戰鬥任務?>
           │  ╱      ╲
           │ 是       否
           │ │        │
           │ ▼        ▼
           │ 觸發戰鬥  執行其他任務
           │ │
           │ ▼

           │
           ├─→ 簡單任務：Lv.1-3 敵人
           ├─→ 中等任務：Lv.4-7 敵人

           └─→ Boss 任務：Lv.10+ 敵人
           │
           ▼
```

**範例 Prompt**:

```
           │   - 玩家等級 (等級越高越少遭遇 -1%/Lv)
           │   - 道具效果 (驅魔香 -50%)
           │
           ▼
[步驟 4] <抽中?>
           │  ╱    ╲
           │ 是     否
           │ │      │
           │ ▼      ▼

           │觸發
           │ │
           ▼ │

           │ (可按 ESC 取消，消耗道具)
```

**遭遇戰特性**:
- 無法完全避免（除非使用特殊道具）
- 敵人等級 = 玩家等級 ±2
- 獎勵較少（80% 正常獎勵）
- 可選擇逃跑（較高成功率）

### 2.1 生成敵人

```javascript
// 敵人生成邏輯
function generateEnemy(playerLevel, difficulty) {
  // 計算敵人等級
  const levelRange = {
    easy: [-2, 0],
    normal: [-1, +1],
    hard: [0, +2],
    boss: [+2, +5]
  }[difficulty];

  const enemyLevel = playerLevel + randomInt(levelRange[0], levelRange[1]);

  // 基礎屬性
  const baseHP = 50 + (enemyLevel - 1) * 20;
  const baseAttack = 10 + (enemyLevel - 1) * 3;
  const baseDefense = 5 + (enemyLevel - 1) * 2;

  // 隨機技能 (1-3 個)
  const skillCount = difficulty === 'boss' ? 3 : randomInt(1, 2);
  const skills = selectRandomSkills(enemyLevel, skillCount);

  return {
    id: generateUniqueId(),
    name: selectEnemyName(enemyLevel, difficulty),
    level: enemyLevel,
    hp: baseHP,
    maxHp: baseHP,
    attack: baseAttack,
    defense: baseDefense,
    skills: skills,
    loot: generateLoot(enemyLevel, difficulty)
  };
}
```

**敵人類型**:

| 類型 | 等級範圍 | HP 倍率 | 技能數 | 特殊能力 |
|-----|---------|---------|--------|---------|
| 史萊姆 | 1-3 | 1.0x | 0-1 | 無 |
| 哥布林 | 2-5 | 1.2x | 1-2 | 閃避 10% |
| 狼人 | 4-7 | 1.5x | 2 | 暴擊 +20% |
| 巨魔 | 6-9 | 2.0x | 2-3 | 再生 5HP/回合 |
| 惡魔 | 8-12 | 2.5x | 3 | 魔法抗性 30% |
| Boss | 10+ | 3.0x+ | 3-5 | 多階段、特殊機制 |

**技能執行流程**:

```javascript
function executeSkill(skill, target) {
  // 1. 檢查 MP
  if (player.mp < skill.mpCost) {
    showError("MP 不足");
    return false;
  }

  // 2. 檢查冷卻
  if (skill.cooldownRemaining > 0) {
    showError("技能冷卻中");
    return false;
  }

  // 3. 扣除 MP
  player.mp -= skill.mpCost;

  // 4. 播放動畫
  playSkillAnimation(skill);

  // 5. 計算傷害
  const baseDamage = skill.power + player.attack;
  const defense = target.defense;
  const critMultiplier = isCriticalHit() ? 1.5 : 1.0;
  const elementMultiplier = getElementMultiplier(skill.element, target.weakness);

  const finalDamage = Math.max(1,
    (baseDamage - defense) * critMultiplier * elementMultiplier
  );

  // 6. 應用傷害
  target.hp -= finalDamage;

  // 7. 顯示結果
  showDamageNumber(finalDamage, target, {
    isCrit: critMultiplier > 1,
    isWeakness: elementMultiplier > 1
  });

  // 8. 設定冷卻
  skill.cooldownRemaining = skill.cooldown;

  // 9. 記錄日誌
  addCombatLog(`${player.name} 使用 ${skill.name}，造成 ${finalDamage} 傷害`);

  return true;
}
```

**傷害計算公式**:

```
最終傷害 = (基礎傷害 - 防禦力) × 暴擊倍率 × 屬性倍率 × 隨機係數

其中：
- 基礎傷害 = 技能威力 + 攻擊力
- 暴擊倍率 = 1.5 (暴擊) 或 1.0 (普通)
- 屬性倍率 = 2.0 (克制) / 1.0 (普通) / 0.5 (抵抗)
- 隨機係數 = 0.85 - 1.15
- 最低傷害 = 1 (保證有傷害)
```

```javascript
function companionAI(companion, battleState) {
  // 1. 評估戰況
  const playerHpPercent = battleState.player.hp / battleState.player.maxHp;
  const enemyHpPercent = battleState.enemy.hp / battleState.enemy.maxHp;

  // 2. 決策優先級
  if (playerHpPercent < 0.3 && companion.hasHealSkill) {
    // 玩家 HP 低，優先治療
    return useHealSkill(companion, battleState.player);
  } else if (enemyHpPercent < 0.2 && companion.hasFinisherSkill) {
    // 敵人 HP 低，使用終結技
    return useFinisherSkill(companion, battleState.enemy);
  } else if (companion.mp > companion.maxMp * 0.5) {
    // MP 充足，使用強力技能
    return useStrongSkill(companion, battleState.enemy);
  } else {
    // 普通攻擊
    return useBasicAttack(companion, battleState.enemy);
  }
}
```

```javascript
function enemyAI(enemy, battleState) {
  // 1. 根據敵人類型選擇策略
  switch (enemy.type) {
    case 'aggressive':
      // 激進型：總是使用最強技能
      return useStrongestSkill(enemy);

    case 'balanced':
      // 平衡型：根據 HP 調整策略
      if (enemy.hp < enemy.maxHp * 0.3) {
        return useDefensiveSkill(enemy);
      } else {
        return useRandomSkill(enemy);
      }

    case 'tactical':
      // 戰術型：針對玩家狀態
      if (battleState.player.hp < battleState.player.maxHp * 0.5) {
        // 玩家 HP 低，猛攻
        return useStrongestSkill(enemy);
      } else if (enemy.hp < enemy.maxHp * 0.5) {
        // 自己 HP 低，治療或防禦
        return useHealOrDefense(enemy);
      } else {
        return useRandomSkill(enemy);
      }

    case 'boss':
      // Boss 型：階段性策略
      return bossBehavior(enemy, battleState);

    default:
      return useBasicAttack(enemy);
  }
}

function bossBehavior(boss, battleState) {
  const hpPercent = boss.hp / boss.maxHp;

  if (hpPercent > 0.7) {
    // 第一階段：試探性攻擊
    return useWeakSkill(boss);
  } else if (hpPercent > 0.3) {
    // 第二階段：全力進攻
    return useStrongSkill(boss);
  } else {
    // 第三階段：狂暴狀態
    return useUltimateSkill(boss);
  }
}
```

```javascript
function calculateEscapeChance(battleState) {
  let chance = 0.7; // 基礎 70%

  // 等級差修正
  const levelDiff = battleState.player.level - battleState.enemy.level;
  chance += levelDiff * 0.1;

  // HP 修正
  const hpPercent = battleState.player.hp / battleState.player.maxHp;
  if (hpPercent < 0.5) {
    chance -= 0.2; // HP 低於 50% 減少 20%
  }

  // 道具加成
  if (battleState.player.hasItem('煙霧彈')) {
    chance += 0.3;
  }

  // 特殊情況
  if (battleState.enemy.isBoss) {
    return 0; // Boss 戰無法逃跑
  }

  if (battleState.isDungeon) {
    chance *= 0.5; // 副本中逃跑難度加倍
  }

  // 限制在 0-1 之間
  return Math.max(0, Math.min(1, chance));
}
```

### 4.1 經驗值計算

```javascript
function calculateExpReward(enemy, battleState) {
  // 基礎經驗值
  const baseExp = 50 + enemy.level * 20;

  // 難度修正
  const difficultyMultiplier = {
    'easy': 0.8,
    'normal': 1.0,
    'hard': 1.3,
    'boss': 2.0
  }[enemy.difficulty];

  // 等級差修正（挑戰高等級敵人獲得更多經驗）
  const levelDiff = enemy.level - battleState.player.level;
  const levelMultiplier = 1 + (levelDiff * 0.1);

  // 回合數修正（快速結束有加成）
  const turnBonus = battleState.turn < 5 ? 1.2 : 1.0;

  // 連勝加成
  const streakBonus = 1 + (battleState.winStreak * 0.05); // 每連勝 +5%

  // 計算最終經驗值
  let finalExp = baseExp * difficultyMultiplier * levelMultiplier * turnBonus * streakBonus;

  // 四捨五入
  return Math.round(finalExp);
}
```

```javascript
function calculateGoldReward(enemy) {
  // 基礎金幣
  const baseGold = 30 + enemy.level * 10;

  // 敵人類型修正
  const typeMultiplier = {
    '史萊姆': 0.8,
    '哥布林': 1.2, // 哥布林喜歡收集金幣
    '巨魔': 1.0,
    '龍': 2.0,     // 龍守護寶藏
    'Boss': 3.0
  }[enemy.type] || 1.0;

  // 隨機波動 (±20%)
  const randomFactor = 0.8 + Math.random() * 0.4;

  // 幸運值加成
  const luckmultiplier = 1 + (player.luck * 0.01);

  // 計算最終金幣
  let finalGold = baseGold * typeMultiplier * randomFactor * luckMultiplier;

  return Math.round(finalGold);
}
```

### 4.3 物品掉落

```javascript
function rollItemDrop(enemy) {
  const drops = [];

  // 掉落表
  const lootTable = enemy.lootTable;

  for (const item of lootTable) {
    const roll = Math.random();

    if (roll < item.dropRate) {
      // 掉落成功
      drops.push({
        id: item.id,
        name: item.name,
        quantity: item.quantity || 1,
        rarity: item.rarity
      });
    }
  }

  // Boss 保證掉落
  if (enemy.isBoss && drops.length === 0) {
    drops.push(enemy.guaranteedDrop);
  }

  return drops;
}
```
