### ✅ 問題 #8: 傷害公式明確化

**修正方案**: 添加完整實例計算

**修改文件**:
- ✅ 03-CORE-MECHANICS.md

**新增內容**:
```
基礎傷害計算：
baseDamage = 100 + (技能MP消耗 × 3) + (玩家等級 × 10)

最終傷害計算：
totalDamage = baseDamage × 親和性倍率 × 弱點倍率 × 抗性倍率

完整範例（Lv.10 玩家使用 15 MP 技能攻擊火屬性敵人）：
- baseDamage = 100 + (15 × 3) + (10 × 10) = 245
- 如果技能是水屬性 → 親和性 1.5 倍
- totalDamage = 245 × 1.5 = 367.5 → 368 傷害
```
