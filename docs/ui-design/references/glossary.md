## 功能與工具

| 技術概念 | RPG 概念 | 說明 |
|---------|---------|------|
| Skill（技能檔） | 魔法技能 | 官方格式的 .md 檔，UI 層映射為 RPG 魔法 |
| Subagent | 戰鬥夥伴（Companion） | 長期陪伴的協作 Agent，整場戰鬥存在 |
| MCP 工具 | 魔法道具 / 魔法書 | 外部工具在圖書館中以「魔法書」形式呈現 |
| Skill Metadata | 技能元數據 | 獨立 JSON 存放 RPG 屬性（MP/冷卻/動畫） |
| Tool（Claude 內建工具） | 魔法 / 咒語 | Read/Write/Edit/Bash 等每個都有對應魔法名稱 |
| 並行工具調用 | 組合魔法 / 連擊 | 多工具同時使用觸發傷害加成 |

## 資源與數值

| 技術概念 | RPG 概念 | 說明 |
|---------|---------|------|
| API 配額 | HP（生命值） | 配額耗盡 = HP 歸零，需要「休息」恢復 |
| 工具調用額度 | MP（魔力值） | 施放技能、召喚夥伴消耗 MP |
| 使用經驗累積 | EXP（經驗值） | 完成任務獲得 EXP，累積可升級 |
| 遊戲內貨幣 | Gold（金幣） | 在商店購買道具、升級技能 |
| Token 消耗 | 成本追蹤（錢莊） | 在商店系統的錢莊中以 Gold 形式呈現 |

## 操作與流程

| 技術概念 | RPG 概念 | 說明 |
|---------|---------|------|
| 對話任務 / Prompt | Quest（任務） | 每次任務型 Prompt 即一場戰鬥 |
| 任務型 Prompt | 遭遇戰觸發 | 包含動作關鍵字的 Prompt 觸發戰鬥 |
| 對話型 Prompt | 酒館對話 | 知識問答類 Prompt 不觸發戰鬥 |
| Plan Mode | 靜止之間 | 時間暫停的規劃室，深度思考空間 |
| AskUserQuestion | 魔法師詢問 | AI 需要用戶決策的互動事件 |
| 錯誤 / 警告 | 敵人特殊攻擊 / 反噬傷害 | 執行失敗映射為戰鬥中的傷害事件 |
| 權限請求 | 解鎖新能力確認 | 文件/系統權限以「新能力」形式呈現 |
| 任務完成 | 戰鬥勝利 | 任務成功完成 = 敵人 HP 歸零 |
| 任務取消 | 逃跑 | 消耗 30% MP，不獲得獎勵 |

## 版本控制

| 技術概念 | RPG 概念 | 說明 |
|---------|---------|------|
| Git Worktree | 平行世界 | 每個 Worktree 是獨立的時間線 |
| main 分支 | 主世界 | 穩定的主分支，高保護 |
| feature 分支 | 冒險世界 | 新功能開發分支 |
| fix 分支 | 修復世界 | Bug 修復分支 |
| experiment 分支 | 實驗世界 | 技術探索分支（7 天過期） |
| hotfix 分支 | 緊急世界 | 緊急修復分支（高優先級） |
| git commit | 版本封印術 | 創建檢查點 |
| git push | 遠程傳送術 | 同步到遠程倉庫 |
| git merge | 融合術 | 合併分支 |
| merge conflict | 時空衝突 | 需要手動解決的合併衝突 |
| stash | 臨時存檔 | 暫存未提交的變更 |

## 戰鬥相關

| 技術概念 | RPG 概念 | 說明 |
|---------|---------|------|
| 任務複雜度 | 敵人等級 | 複雜度分數 = 敵人 Lv |
| 複雜度 < 8 | 簡單任務（不觸發戰鬥） | 主 CLI 直接完成 |
| 複雜度 ≥ 8 | 遭遇戰 | 觸發戰鬥，生成敵人 |
| 自動戰鬥 | Claude 行動 | AI 使用工具自動映射為攻擊 |
| 手動技能 | 用戶主動觸發 | /commit, /test 等 Skill 命令 |
| 非同步任務 | 後台戰鬥 | 複雜任務在後台獨立執行 |
| 並發任務 | 多重戰鬥 | 最多 3 個並發戰鬥（自動 Worktree 隔離） |
| 任務完成度 | 敵人 HP 減少 | 每個工具調用都推進戰鬥進度 |
