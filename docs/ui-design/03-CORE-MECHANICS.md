## 一、資源系統

### HP（生命值）

| 屬性 | 說明 |
|------|------|
| 對應概念 | API 配額 |
| 初始值 | 100 |
| 最大值 | 隨等級成長 |
| 探索模式恢復 | +2 HP/秒 |
| 戰鬥模式恢復 | 0（不恢復） |
| 歸零效果 | 配額耗盡，需要「休息」等待恢復 |

### MP（魔力值）

| 屬性 | 說明 |
|------|------|
| 對應概念 | 工具使用資源 |
| 初始值 | 100 |
| 最大值 | 隨等級成長 |
| 探索模式恢復 | +1 MP/秒 |
| 戰鬥模式恢復 | +0.1 MP/秒（極慢） |
| 消耗來源 | 技能施放、召喚、Worktree 操作 |

### EXP（經驗值）

| 屬性 | 說明 |
|------|------|
| 對應概念 | 累積使用經驗 |
| 獲得來源 | 完成戰鬥、使用技能、合併 Worktree |
| 每次技能使用 | +10-30 EXP（依技能等級） |
| 戰鬥勝利 | +50-1200 EXP（依敵人等級） |

### Gold（金幣）

| 屬性 | 說明 |
|------|------|
| 對應概念 | 遊戲內貨幣 |
| 獲得來源 | 完成戰鬥、成就獎勵 |
| 用途 | 商店購買道具、升級技能 |

## 二、等級成長系統

### 升級公式

```
升級所需 EXP = 100 × (當前等級)^1.5

範例：
  Lv.1 → Lv.2: 100 EXP
  Lv.2 → Lv.3: 283 EXP
  Lv.5 → Lv.6: 1,118 EXP
  Lv.10 → Lv.11: 3,162 EXP
  Lv.20 → Lv.21: 8,944 EXP
```

### 升級獎勵

每次升級：
- HP 上限 +10
- MP 上限 +5
- 解鎖新技能（特定等級）
- 解鎖新區域（特定等級）

### 等級解鎖表

| 等級 | 解鎖內容 |
|------|---------|
| Lv.1 | 基礎技能（讀心術、創造術、改寫術） |
| Lv.3 | 版本封印術（commit）、森林區域 |
| Lv.5 | 夥伴召喚、試煉之法（test）、山脈區域 |
| Lv.8 | 蟲洞副本、工匠鋪 |
| Lv.10 | 部署魔法、荒原區域 |
| Lv.12 | 架構迷宮副本、組合技系統 |
| Lv.15 | 重構神通、火山區域、安全地城副本 |
| Lv.20 | 多模型切換、非同步戰鬥 |

## 三、冷卻系統

每個技能使用後進入冷卻期，冷卻期間無法再次使用該技能。

### 冷卻時間分級

| 技能類型 | 冷卻時間 | 範例 |
|---------|---------|------|
| 基礎工具 | 0 秒 | 讀心術（Read）、偵測術（Grep） |
| 基礎技能 | 30-60 秒 | 版本封印術（commit） |
| 進階技能 | 90-120 秒 | 部署魔法（deploy）、重構神通 |
| 召喚夥伴 | 300 秒（5 分鐘） | 所有 Subagent 召喚 |

### 冷卻規則

- 探索模式和戰鬥模式中冷卻速度相同（正常倒數）
- 在玩家住所「休息」可重置所有冷卻
- 組合技有獨立的冷卻計時器

## 四、組合技系統

連續在指定時間視窗內使用特定技能組合，可觸發組合技獎勵。

### 組合技列表

| 組合技名稱 | 所需技能 | 時間視窗 | 獎勵倍率 |
|-----------|---------|---------|---------|
| 完美代碼連擊 | 代碼生成術 → 代碼審查術 → 測試生成術 | 30 秒 | EXP ×2.0、Gold ×2.0 |
| 調試連環斬 | 讀心術 → 偵測術 → 改寫術 | 20 秒 | EXP ×1.5 |
| 版本大師 | 創造術 → 版本封印術 → 部署魔法 | 60 秒 | Gold ×3.0 |
| 並行魔法 | 同時使用 2+ 個工具（自動偵測） | 即時 | EXP ×1.5 |

### 組合技觸發規則

1. 技能必須在時間視窗內**依序**使用（並行魔法除外）
2. 每個技能之間可以穿插其他操作，不中斷計時
3. 組合技觸發後有獨立冷卻（通常為組成技能冷卻總和的 50%）
4. 觸發時顯示特殊動畫和獎勵提示

## 五、傷害計算系統

### 基礎傷害公式

```
傷害 = 技能基礎威力 × 屬性相性倍率 × (1 + 隨機浮動)

其中：
- 技能基礎威力：每個技能固定值（如 Read = 10, Write = 25, Bash = 30）
- 屬性相性倍率：見下方屬性相性表
- 隨機浮動：-10% ~ +10%
```

### 敵人 HP 計算

```
敵人 HP = 50 × 複雜度等級

範例：
  複雜度 5 → HP 250
  複雜度 10 → HP 500
  複雜度 15 → HP 750
```

### 屬性相性表

每個技能有「屬性」，每個敵人有「弱點」和「抗性」：

| 技能屬性 | 克制（×1.5） | 被克制（×0.5） |
|---------|-------------|---------------|
| 🔥 火（攻擊型：Write, Bash） | 🌿 木（結構類敵人） | 💧 水（邏輯類敵人） |
| 💧 水（分析型：Read, Grep） | 🔥 火（Bug 類敵人） | 🌿 木（資料類敵人） |
| 🌿 木（建構型：Edit, Glob） | 💧 水（性能類敵人） | 🔥 火（安全類敵人） |
| ⚡ 雷（組合型：並行操作） | 所有屬性 ×1.2 | 無弱點 |

## 六、成就系統

追蹤玩家行為，解鎖成就並獲得獎勵。

### 成就分類

| 類型 | 範例 | 獎勵 |
|------|------|------|
| 首次行動 | 首次使用技能、首次戰鬥勝利、首次召喚夥伴 | EXP + 稱號 |
| 累計里程碑 | 使用技能 100 次、戰鬥勝利 50 次 | Gold + 特殊道具 |
| 組合技 | 觸發 5 次組合技、觸發所有組合技 | EXP ×2 + 稱號 |
| 堅持不懈 | 連續使用 7 天、連續使用 30 天 | 特殊召喚獸解鎖 |
| 探索 | 訪問所有場所、完成所有副本 | 隱藏區域解鎖 |
