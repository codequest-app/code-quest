## 文件目的

本文件定義 CodeLand 世界的完整地圖結構、場所設計和 UI 呈現方式。包含 3 大區域、14+ 個場所的詳細界面設計和互動規格。

## CodeLand 世界觀

### 世界構成

Code

Quest 的世界稱為 **CodeLand**，由三大區域組成：

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│                      🏔️ 副本區域                            │
│                   (Dungeon Zone)                            │
│          特殊任務、Boss 戰、完成前無法退出                    │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                    🌲 野外區域                               │
│                  (Wilderness Zone)                          │
│        危險區域、隨機遭遇戰、執行代碼任務的主要場所             │
│                                                             │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│                     🏰 城鎮區域                              │
│                     (Town Zone)                             │
│           安全區、商店、NPC、休息補給、管理中心               │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 區域特性對比

| 特性 | 城鎮區域 | 野外區域 | 副本區域 |
|-----|---------|---------|---------|
| **安全性** | 安全 ✅ | 危險 ⚠️ | 危險 ⚠️⚠️ |
| **遭遇戰** | 無 | 隨機觸發 | 固定觸發 |
| **可逃離** | N/A | 可以 | 不可以 |
| **資源恢復** | 快速 | 無 | 無 |
| **商店訪問** | 可以 | 不可以 | 不可以 |
| **Worktree 操作** | 可以 | 不可以 | 不可以 |

| Stash 應用 | 任何 | 3 MP | 無 | 應用保存的變更 |

**重要說明**：
- Worktree 操作只能在探索模式執行
- 戰鬥中無法操作 Worktree
- 所有 MP 消耗來自全局 MP 池

## 場所切換與動畫

### 城鎮內場所切換

**走路進入（慢速）**:
```
時間軸:
0.0s  玩家走到建築門口
0.0s  顯示「按 [E] 進入」提示
用戶按 E
0.0s  淡出當前畫面 (300ms)
0.3s  顯示「進入中...」(200ms)
0.5s  淡入新場所 (300ms)
0.8s  完成
```

**快捷鍵進入（快速）**:
```
時間軸:
用戶按快捷鍵 (例如: 1)
0.0s  閃光效果 (100ms)
0.1s  粒子爆發 (100ms)
0.2s  淡入目標場所 (100ms)
0.3s  完成
```

### 區域間切換

**城鎮 ↔ 野外**:
```
時間軸:
0.0s  淡出當前畫面 (300ms)
0.3s  旋轉粒子效果 (200ms)
0.5s  淡入新區域 (300ms)
0.8s  完成

視覺效果:
• 城鎮: 藍綠色調 → 野外: 自然色調
• 背景音樂切換
• 遭遇率指示出現（野外）
```

**野外 → 副本**:
```
時間軸:
0.0s  確認進入副本
0.0s  畫面震動 (200ms)
0.2s  紅色閃光 (300ms)
0.5s  淡出到黑色 (500ms)
1.0s  淡入副本入口 (500ms)
1.5s  顯示副本資訊彈窗
用戶點擊「挑戰副本」
2.0s  副本戰鬥開始

視覺效果:
• 暗色調、危險氛圍
• 特殊音效（陰森、緊張）
• 「無法退出」警告標誌
```
