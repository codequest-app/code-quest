## 世界觀

你是一名代碼魔法師，居住在 CodeLand 的世界中。這個世界由三大區域組成：城鎮（安全區）、野外（危險區）、副本（特殊任務區）。

## 一、城鎮區域（Town Zone）— 安全區

城鎮是冒險者的根據地。不會觸發戰鬥，提供各種服務設施。

### 1.1 商業街（Shopping District）

**位置**：城鎮中心偏東
**功能**：包含 7 個商店，提供技能、工具、夥伴管理

```
商業街
├─ 🔮 技能商店     — 查看、解鎖、升級技能
├─ ⚒️ 工匠鋪       — 創建自定義技能（6 步驟引導）
├─ 📚 魔法圖書館   — 瀏覽和安裝 MCP 工具
├─ 🧙 傭兵公會     — 管理戰鬥夥伴（Subagents）
├─ 🏆 寶物庫       — 成就、召喚獸圖鑑、收集品
├─ 🎯 訓練場       — 教學、模擬戰鬥（不消耗資源）
└─ 💰 錢莊         — 資源統計、AI 模型成本追蹤
```

**限制**：戰鬥中無法進入；探索模式隨時可訪問；支持快捷鍵 1-7 直接進入對應商店。

→ 詳見 [systems/06-shop-system.md](systems/06-shop-system.md)

### 1.2 酒館（Tavern）

**位置**：城鎮中心偏西
**功能**：AI 對話、日常討論、技術諮詢

酒館是對話型 Prompt 的主要處理場所。在這裡與 AI 對話不消耗 MP，不觸發戰鬥。如果對話涉及複雜任務，AI 會建議前往靜止之間制定計劃或前往野外執行。

**NPC**：酒館老闆（AI 助手）— 友善的 AI，回答各種問題

**限制**：任何時候都可進入（副本中除外）；不能在酒館觸發戰鬥。

### 1.3 靜止之間（Stasis Chamber）

**位置**：城鎮最北端（獨立建築）
**功能**：Plan Mode 專屬空間，深度思考和戰術規劃

概念類似「精神時光屋」— 進入後外界時間暫停，可以無限時間制定計劃。

**使用場景**：
- 戰鬥前規劃：遭遇強敵時 AI 建議進入此處制定戰術
- 複雜任務規劃：AI 偵測到需要 Plan Mode 時自動引導
- 主動深度思考：玩家自行前往進行深入討論

**台詞**：進入時「時間在此靜止，唯有你的思緒在流動。」離開時「帶著你的計劃回到世界吧。」

**限制**：Plan Mode 只能在此處觸發；探索模式隨時可進入；戰鬥中可暫停進入。

→ 詳見 [systems/07-interactive-events.md](systems/07-interactive-events.md)

### 1.4 公會大廳（Guild Hall）

**位置**：城鎮中心
**功能**：Worktree 管理中心（平行時間線管理）

管理所有「平行世界」（Git Worktrees）。查看正在進行的戰鬥分支，手動創建 / 切換 / 合併 Worktrees。

**NPC**：公會管理員 — 協助 Worktree 管理；任務公告板 — 顯示正在進行的戰鬥列表

**限制**：探索模式隨時可進入；戰鬥中無法操作 Worktree（但可查看）。

→ 詳見 [systems/08-worktree-system.md](systems/08-worktree-system.md)

### 1.5 玩家住所（Player Home）

**位置**：城鎮南部
**功能**：休息、存檔、設定

在住所「休息」可完全恢復 HP/MP。此外可手動存檔、調整 RPG 化程度、音效開關、快捷鍵設定、AI 模型選擇。

## 二、野外區域（Wilderness Zone）— 危險區

野外是代碼任務的執行地，會觸發遭遇戰。完成戰鬥可獲得經驗值和金幣。

### 子區域總覽

| 區域 | 遭遇率 | 複雜度範圍 | 建議等級 | 說明 |
|------|--------|-----------|---------|------|
| 🌳 森林（Forest） | 30% | 5-8 | Lv.5-10 | 入門級任務區 |
| ⛰️ 山脈（Mountains） | 50% | 8-12 | Lv.8-15 | 中級任務區 |
| 🏜️ 荒原（Wasteland） | 70% | 12-15 | Lv.12-20 | 高級任務區 |
| 🌋 火山（Volcano） | 90% | 15+ | Lv.18+ | 專家級任務區 |

### 遭遇戰觸發邏輯

在野外時，依以下條件判斷是否觸發戰鬥：

1. **Prompt 類型**：必須是任務型（包含動作關鍵字如修復、創建、重構…）
2. **複雜度門檻**：分析複雜度 ≥ 8 才觸發戰鬥；< 8 的簡單任務由主 CLI 直接完成
3. **區域影響**：不同子區域的遭遇率影響隨機遭遇機率

觸發戰鬥後會自動創建 Worktree 作為戰鬥分支，戰鬥結束可選擇合併 / 保留 / 放棄。

### 移動規則

- 城鎮 ↔ 野外：雙向自由切換（非戰鬥時）
- 戰鬥中：無法切換場所（可暫停進入靜止之間）
- 可隨時返回城鎮（非戰鬥時按 Esc 或快捷鍵 B）

## 三、副本區域（Dungeon Zone）— 特殊任務區

副本是特定複雜任務的專屬區域，包含劇情戰鬥和 Boss 戰。進入後無法退出，需完成戰鬥才能離開。

### 副本列表

| 副本 | 類型 | Boss | 特殊機制 | 建議等級 |
|------|------|------|---------|---------|
| 🐛 蟲洞（Bug Cave） | Debug 任務 | Bug 之王 | 追蹤錯誤日誌 | Lv.8-12 |
| 🏗️ 架構迷宮（Architecture Maze） | 架構重構 | 遺留代碼守護者 | 制定完整計劃 | Lv.12-18 |
| 🧪 測試試煉場（Testing Arena） | 測試覆蓋 | 測試覆蓋監督者 | 達到 100% 覆蓋率 | Lv.10-15 |
| 🔒 安全地城（Security Dungeon） | 安全漏洞修復 | 漏洞注入魔王 | 通過安全檢查 | Lv.15-20 |

### 觸發方式

- **特定 Prompt 觸發**：偵測到對應關鍵字 + 高複雜度時，AI 提示進入副本並要求確認
- **主動進入**（未來擴展）：在城鎮的副本入口查看資訊後選擇挑戰

### 副本規則

- 進入後無法返回城鎮（完成後自動返回）
- 無法在副本中創建新 Worktree（已綁定戰鬥分支）
- 可以使用道具和召喚夥伴
- 可以在 Boss 戰前進入靜止之間

## 場所轉換規則

```
城鎮 ←──→ 野外     雙向，非戰鬥時隨時可切換
城鎮  ──→ 副本     單向進入，完成後自動返回城鎮
野外  ──→ 副本     特定 Prompt 觸發
副本  ──→ 城鎮     戰鬥完成後自動返回

戰鬥中：
  ✗ 無法切換場所
  ✓ 可進入靜止之間（時間暫停）
  ✓ 可在酒館對話（時間暫停）
```
